# Contributing

This repo has one rule: **claims need evidence.**

- A new strategy/indicator starts as `⚠️ Untested`. It earns `✅`/`❌`/`🔶` only after going through the gauntlet in [METHODOLOGY.md](METHODOLOGY.md) (no-look-ahead, cost-internalized, placebo, OOS, DSR, PBO).
- Use [`deflate`](https://github.com/raphael2025/deflate) to validate; link the reproducible test.
- **Synthesize and cite — never paste** others' copyrighted text. Link the source.
- Follow [TEMPLATE.md](TEMPLATE.md). Keep all three languages (en/zh/ru) in sync; if you can only do one, open the PR anyway and tag for translation.
- No hype, no "guaranteed", no signals-for-sale. Honest verdicts only.
