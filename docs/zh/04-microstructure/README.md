# 04 · 市场微观结构

从订单簿和成交带读出的信号——净压力、毒性、排队动态——评估的是 *零售吃单者* 在扣除价差、费用和延迟后能否真正捕获它们。

| 信号 | 判决 | 原因 |
|---|---|---|
| [订单流失衡 / VPIN](order-flow-imbalance.md) | 🔶 | 亚秒级视野下真实 RankIC ≈ 0.29——但这是仅属于做市/HFT 的优势;零售吃单者跨越价差后净得 ≈ −8 bps。 |

微观结构是唯一一个信号常常 **真正具备预测力** 的家族——然而这并不是要紧的问题。这些优势活在 *价差之下、延迟地板之下*:捕获它们意味着提供流动性并在微秒级行动,因此它们属于流动性提供者,而非零售吃单者。
