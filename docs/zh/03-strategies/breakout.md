# 突破

> 一句话:当价格突破前期区间高点或跌破区间低点时入场——唐奇安/通道突破与波动率挤压突破。**判决:🔶 有条件**。

## 它是什么

突破策略等待价格越过一个参考水平——前 N 根 K 线的最高高点(唐奇安通道)、价格通道的边缘,或在一段波动率压缩后的上/下轨(一次"挤压")。该突破被视为新方向性行情已开始的信号,你顺着突破方向入场。

## 主张

区间代表均衡;一次决定性突破意味着平衡被打破、新趋势正在启动。在突破处入场,你就能在人群之前从行情起点捕捉到它。波动率挤压版本还加了一个择时故事:窄轨意味着蓄势待发,随之而来的扩张就是这笔交易。

## 检验

我们在加密永续合约(BTC/ETH/SOL)上评估突破入场,以 K 线收盘价标注时间戳(无未来函数),并将 **taker 费用 + 滑点 + 资金费内化**,且刻意区分两个种类:

- **(a) 缓慢通道突破。** 突破一个长而慢的唐奇安/通道水平,机械上就是一次*趋势跟随入场*——当以缓慢方式交易并按波动率定仓时,它继承了同样温和、可覆盖成本的优势(Sharpe 量级 ~1.1)。
- **(b) 快速/日内"追突破"。** 在低周期上,它被**假突破**(价格戳一下水平、吸入入场单,然后反转)和**换手成本**(约 −86%/年 的量级)所主导。净亏损;相对安慰剂无优势;在水平/回看窗口搜索下 DSR 崩塌。

## 判决:🔶 有条件

突破*仅*作为缓慢、成本感知的变体时为真——而在那一刻,它本质上就是换了件外衣的[趋势跟随](trend-following.md),并因相同的原因而幸存(它骑乘持续性、覆盖成本、坐落在参数高原上)。朴素的日内突破是 ❌:假突破加换手把它们流血放干。**波动率挤压 → 突破**这一情形是真正有条件的那种——当随后的行情缓慢且趋势型时它能奏效,但它并非免费;作为快速的轨道扩张触发器来检验时,它的表现与其余[布林带](../01-technical-indicators/bollinger-bands.md)家族成员一样。归入 ✅ 一侧的条件:缓慢地交易它,接受与趋势跟随相同的肥尾/低胜率盈亏分布,并且永远不要把它收紧成一个日内信号。

## 自己试试

在 K 线收盘价上构建两个版本——一个缓慢通道突破和一个快速日内突破——施加现实的每次换手成本,然后把两者都跑过 [`deflate`](https://github.com/raphael2025/deflate):`deflated_sharpe`(惩罚回看窗口/水平网格)、`pbo` 和 `placebo`。缓慢版本应能通过这道关卡(并且看起来很像趋势跟随);快速版本则在成本和安慰剂上失败。这个对比*就是*判决。
