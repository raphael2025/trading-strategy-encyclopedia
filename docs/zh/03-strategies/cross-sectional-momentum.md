# 横截面动量

> 一句话:按近期收益(结合一个非流动性因子)对一个流动的标的池排名,做多顶部分位、做空底部分位——相对动量,而非绝对动量。**判决:✅ 已验证**(在正确的标的池上)。

## 它是什么

趋势跟随问的是"*这个*资产在上涨吗?",而横截面动量问的是"哪些资产*相对彼此*在上涨?"你取一个流动的标的池,按每个名字在某个回看窗口内的尾随收益给它们排名,做多最强的分位(比如前 20%)、做空最弱的(后 20%),然后定期再平衡。加入一个**非流动性**因子——向那些收益尚未被套利抹平的名字倾斜——会让这个价差更锐利。

## 主张

赢家继续赢、输家继续输,这在横截面上、在中期跨度上成立。产生时间序列趋势的同一种反应不足与羊群效应,产生了一个持续足够久、可通过持有两端之间价差来收割的相对排名——一种近乎市场中性的押注,不需要整个市场上涨。

## 检验

我们对一个**流动的标的池**排名,交易顶部/底部 20%,把**动量 + 非流动性**组合成一个多因子分数,以再平衡收盘价标注时间戳(无未来函数),并将**费用 + 滑点 + 资金费内化**,且以选择性偏差感知的方式对因子/回看窗口的选择进行评分。

- **Sharpe ≈ 1.2**,通过 Deflated Sharpe 和 PBO——一个真实、分散的优势。
- **依赖标的池。** 在股票**大市值**上,同样的因子是 ❌(DSR ≈ 0.004)——大市值被套利得太有效率;这一溢价存在于中小市值,以及加密市场中流动但被套利较少的角落。
- **完美地组合。** 与[趋势跟随](trend-following.md)和资金费 carry 叠加后,该混合组合达到 **Sharpe ≈ 1.7、DSR 0.93、回撤 ≈ −22%**——因为这三个分仓的相互相关性很低。

## 判决:✅ 已验证

它之所以幸存,是因为它在许多名字上分散、*相对*(因此不依赖市场方向)、足够慢以覆盖成本,且具备足够的选择性偏差感知以通过 DSR/PBO。条件:它只在正确的**标的池**上奏效——在超高效率的大市值中,这一优势在结构上是缺失的(DSR ≈ 0.004),因此一个流动但未饱和的标的池是必需的。现实预期:约 1.2 的独立 Sharpe——这一点和趋势跟随一样,单凭自身也只是勉强超过等权买入持有基准(Sharpe ~1.11)。它真正的价值在于它给组合账本带来的**低相关性**;组合分仓见[判决索引](../00-verdict-index.md)。

## 自己试试

按尾随收益加一个非流动性倾斜对一个流动的标的池排名,构建一个做多前 20% / 做空后 20% 的组合,以现实成本再平衡,然后跑过 [`deflate`](https://github.com/raphael2025/deflate):`deflated_sharpe`(惩罚因子/回看窗口搜索)、`pbo`、`placebo`。然后在一个仅含大市值的标的池上重跑,看着 DSR 朝 ~0 崩塌——这个对比表明,这一优势是依赖标的池的,而非普适的。
