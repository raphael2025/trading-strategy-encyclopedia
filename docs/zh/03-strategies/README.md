# 03 · 策略

完整的交易*策略*——入场逻辑、持仓规则、出场纪律——在诚实、因果的时间戳标注下进行检验,成本内化、且具备选择性偏差感知的紧缩(DSR/PBO),而非孤立地看单个指标。

| 策略 | 判决 | 原因 |
|---|---|---|
| [趋势跟随](trend-following.md) | ✅ | 以动量真实存在的方式骑乘其持续性;足够慢以覆盖成本,具备选择性偏差感知。Sharpe ~1.1,胜率 ~36% 但盈亏比 ~6.5——优势在于*不*砍掉盈利单。 |
| [横截面动量](cross-sectional-momentum.md) | ✅ | 相对、分散、低相关;Sharpe ~1.2 且通过 DSR/PBO——但仅在流动但未饱和的标的池上成立(大市值股 DSR ≈ 0.004)。 |
| [突破](breakout.md) | 🔶 | 仅作为缓慢、成本感知的通道突破时为真(本质上就是趋势跟随);朴素的日内"追突破"会死于假突破 + 换手成本。 |
| [均值回复](mean-reversion.md) | ❌ | 在加密市场的趋势型环境中,逆向博弈极端值 = 接飞刀;计入成本后净亏损,相对安慰剂无优势。(仅在市场中性配对/统计套利中合法。) |
| [跟单](copy-trading.md) | ❌ | RankIC ≈ 0、幸存者偏差,而且你在领头交易者之后成交、充当他们的退出流动性——一个对未来零信息含量的排名不是策略。 |

幸存下来的策略都做着同一件事:它们以缓慢、成本感知、选择性偏差感知的方式骑乘**动量/持续性**,并分散进入**低相关组合**(趋势 + 横截面 + carry 可达 Sharpe ~1.7)。死掉的那些要么**对抗趋势**(均值回复),要么**追逐海市蜃楼**(跟单 = 充当别人的退出流动性)。
