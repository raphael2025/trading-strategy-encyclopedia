# 谐波形态

> 一句话:基于斐波那契比例的反转形态——Gartley、Bat、Butterfly、Crab、Shark、AB=CD——它们标记出一个"潜在反转区"。**判决:❌(曲线拟合 + 未来函数 + 多重检验)。**

## 是什么

谐波形态是一类多段式价格结构(通常标记为 X-A-B-C-D),只有当各段命中特定的斐波那契比例时才算"成立":

| 形态 | 关键比例 |
|---|---|
| **Gartley** | AB=CD,0.618 回撤 |
| **Bat** | 0.886 XA 回撤 |
| **Butterfly** | 1.272 / 1.618 扩展 |
| **Crab** | 1.618 扩展 |
| **Shark** | 0.886、1.13 |
| **AB=CD** | 1.0 / 1.272 / 1.618 |
| **Three Drives** | 1.272、1.618 |

这些比例汇聚之处便是**潜在反转区(Potential Reversal Zone, PRZ)**;交易方式是在该处反向操作(fade),预期价格反转,止损就设在 PRZ 之外不远处。

## 它声称什么

声称市场以斐波那契比例的波段运行,因此当某个结构在容差范围内匹配上这些比例时,PRZ 便是一个高概率的转折点。支持者(Gartley 的 *Profits in the Stock Market*,以及后来 Scott Carney 的 *Harmonic Trading*)将其呈现为精确的、基于规则的入场——因为比例"客观",所以比徒手画出的形态更严谨。

## 检验

这种"客观性"是一种错觉,而该结构是教科书级别的过拟合机器:

- **容差带 = 自由参数。**"0.618 ± 容差"、"1.618 ± 容差",再加上对哪些波段算作 X/A/B/C/D 的选择,赋予了极大的自由裁量空间。有了这么多自由度,你几乎可以在任何图表上事后找到一个"有效"的谐波形态——而任何如此灵活的东西都会过拟合历史,并在样本外失效。
- **未来函数。**点 D(入场点)只有在结构完成之后才可知;标记这些波段需要看到未来,这一点与[头肩形态](head-and-shoulders.md)和[艾略特波浪](elliott-wave.md)如出一辙。
- **多重检验。**七种以上命名的形态 × 比例容差 × 时间周期,这保证了样本内的赢家;**Deflated Sharpe** 会把整个形态族系压缩为噪声。没有任何大样本、经同行评审的证据表明谐波入场在扣除成本后能打赢[匹配的安慰剂检验](../../METHODOLOGY.md)——而 KB 自己的来源(Bulkowski)根本没有提供任何谐波专属的失败统计,只有从二手参考资料里搬来的比例。
- **斐波那契没有机制。**没有任何有据可查的理由说明价格*应该*在 0.886 而非 0.85 处反转;这是披着几何外衣的数字命理学。

## 判决:❌

谐波形态集齐了本仓库收录的每一种失败模式:事后的波段标记、能拟合任何东西的容差带、一大堆诱导挑选的形态,以及没有因果机制——再加上加密市场偏高的假反转率和交易成本。没有可信的证据表明它存在样本外、扛得住成本、能打赢安慰剂的优势。PRZ 最大的用处是当作一份拥挤价位的清单,这些价位会被[猎杀](../07-market-mechanics/liquidity-hunts.md),而不是当作一个可靠的转折点。

## 自己动手试

锁定单一形态,使用*固定*的比例容差和一个客观的波段探测器,在 D 点确认时给入场打上时间戳(不许偷看),把成本内化,然后用 [`deflate`](https://github.com/raphael2025/deflate) 在整个形态族系上运行 `placebo` + `deflated_sharpe`。当你尝试过的形态/容差数量被惩罚后,看着 DSR 崩塌。
