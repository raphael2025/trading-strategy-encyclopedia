# 头肩形态

> 一句话:一种三峰反转形态(左肩、更高的头、右肩),据称价格跌破"颈线"时趋势反转。**判决:❌ 已证伪。**

## 它是什么

头肩形态(H&S)是技术分析中最著名的反转形态。在上升趋势中,它画成三个峰:一个 **左肩**、一个更高的 **头**,以及一个与左肩大致齐平、较低的 **右肩**。连接中间两个谷底的趋势线就是 **颈线**。规则是:当价格收盘跌破颈线,上升趋势便结束,**价格目标** 为从头到颈线的垂直高度,自突破点向下投射。其镜像——**倒头肩**——是看涨的筑底版本(三个谷底,中间最深,向上突破颈线)。

## 主张

头肩形态是一个 *高概率* 反转信号:完成的形态加上颈线突破,可靠地标志趋势改变,并带有一个可测量、可交易的目标。在零售资料和形态百科中,广泛流传着 70–80%+ 的"胜率"数据。

## 检验

关键问题在于 **你被允许在何时知道这个形态存在。** 右肩只有在 *之后* 的K线确认它停止上涨后才算右肩;"头"只有在两侧都有K线后才算头;颈线只有在两个谷底都打印出来后才能画出。因此,唯一在因果上诚实的交易是 **在确认的颈线突破处入场,且每个摆动点都按真正确认它的那根K线打时间戳——绝不事后追溯。**

我们在加密永续合约(BTC/ETH/SOL)和股票上用算法检测头肩形态(摆动点枢轴,以肩部对称容差和头部最小突出度为参数),并:

- **无未来函数。** 枢轴只有在所需的向前观察K线之后才确认;入场在颈线突破那根K线的收盘处。
- **内化成本**(吃单/taker 费用 + 滑点 + 资金费)。
- **匹配安慰剂:** 突破后收益与按时段和当前动量匹配的随机入场对照。

发现:

- **事后追溯造成的虚高就是全部效应。** 用未来信息标记形态(教科书做法)会产生那些著名的高胜率。一旦因果对齐打时间戳,突破后收益与匹配安慰剂 **无法区分**,扣净成本后持平甚至为负。
- **无限可调参 = 多重检验。** 肩部要多对称?头部要多深?颈线斜率多少?每一个旋钮都是一个搜索维度;样本内总有东西看起来不错。**Deflated Sharpe Ratio** 惩罚家族规模并向零收缩;**PBO** 很高。

## 判决:❌ 已证伪

致命因素是 **未来函数。** 头肩形态在文献中"有效",是因为形态是在事后识别的,此时反转早已发生——这种识别借用了答案。一旦把摆动点和颈线按诚实的确认时刻打上时间戳,相对匹配安慰剂入场便没有优势,成本再补上致命一击。跨形态可调参定义的多重检验则锁定了判决。这与 [SMC / 区域家族](../00-verdict-index.md) 的其余成员是同一种失效模式:供需区、FVG 回踩和流动性扫荡全都死于因果时间戳加安慰剂。另见 [K线形态](candlestick-patterns.md),那里的主导致命因素转为多重检验。

## 自己试试

用固定的枢轴规则检测头肩形态,每个摆动点只在其向前观察窗口之后确认,在颈线突破收盘处入场,并扣净真实成本。然后让它跑一遍 [`deflate`](https://github.com/raphael2025/deflate):`placebo`(对照匹配的随机入场)、`deflated_sharpe`(惩罚你搜索过的对称度/深度/斜率网格)以及 `pbo`。最先垮掉的是 placebo——一旦时点诚实,该形态就不携带安慰剂所没有的任何信息。
