# 三角形、旗形与楔形

> 一句话:整理形态——上升/下降/对称三角形、旗形/三角旗、上升/下降楔形——作为持续或突破信号来交易。**判决:❌(未来函数 + 多重检验 + 成本);缓慢的通道突破是唯一的有条件幸存者。**

## 是什么

这些是经典的"图表正在盘整"形态:

- **三角形**——收敛的趋势线。*上升三角形*:顶部水平,低点抬高。*下降三角形*:底部水平,高点下移。*对称三角形*:两条线都收敛("盘卷")。
- **旗形 / 三角旗**——在一段急速行情之后出现的小型逆势通道(旗形)或微型三角形(三角旗),被解读为持续行情前的一次停顿。*高位紧凑旗形*:在两个月内约 90–100% 的飙升之后形成。
- **楔形**——两条线朝同一方向倾斜。*上升楔形*(通常看跌)、*下降楔形*(通常看涨)。

交易方式是在预期方向上的突破处入场,价格目标按形态高度测算得出。

## 它声称什么

压缩储存了"能量";突破把它释放为一段可测量的行情。Bulkowski 的 *Encyclopedia of Chart Patterns*(第 3 版,2021)是被引用最多的统计来源,它在一个大型美股样本上对约 39 种形态进行了排名,并报告了诸如此类的数据:上升三角形向上突破失败率约 17%,平均涨幅约 43%;矩形底失败率约 15%,平均涨幅约 48%;对称三角形排名 **36/39**("表现糟透了");上升楔形向下突破失败率 **51%**;三角旗失败率 **约 54%**,平均行情仅约 7%。

## 检验

注意 Bulkowski 自己的数字已经说明了什么:各形态的平均失败率约为 22–25%(每 4–5 次中 1 次),而*最差*的那几个常见形态(对称三角形、楔形、三角旗)就是掷硬币。但更深层的问题在于*这些统计是如何产生的*,以及在我们的层层关卡下会发生什么:

- **"完美交易" / 事后挑选。**Bulkowski 的统计使用的是形态干净、经手工确认的样本。在真实时间里,你无法知道一个三角形是"有效"的(5 次触碰、无空白),直到它已经走完——这就是[未来函数偏差](../../../METHODOLOGY.md),与[头肩形态](head-and-shoulders.md)一样的杀手。
- **多重检验。**数十种命名的、可调参的形态 × 回看窗口,保证了某些在样本内看起来很棒。用 **Deflated Sharpe** 惩罚整个族系,优势便会被压缩到趋近于零——意识到挑选问题的检验是决定性的。
- **假突破 + 成本。**加密市场的假突破率高达 **35–50%**(相比之下股市约 20–25%),这归因于杠杆、割裂的流动性,以及巨鲸猎杀那个显而易见的价位。在日内用吃单/taker 手续费 + 滑点来交易,净结果是负的;这正是为什么快速[突破](../03-strategies/breakout.md)交易是 ❌。

## 判决:❌(附一项有条件的豁免)

作为离散的、手工画出的信号,这些形态失败了:用于验证的统计是事后视角,形态动物园诱导过拟合,而加密市场的假突破率加上换手成本给了它们最后一击。**唯一**能幸存的版本是*作为[趋势跟随](../03-strategies/trend-following.md)来交易的缓慢通道/唐奇安(Donchian)突破*——它属于 🔶 有条件,而且它管用是出于趋势持续性的原因,并非因为三角形"预测"了什么。不要交易形状;如果你要交易突破,就慢慢交易、按波动定仓、并意识到挑选问题。

## 自己动手试

用*因果*方式定义单一形态(例如,唐奇安突破——一个完全客观的三角形/通道代理),设两种速度:缓慢和日内。把成本内化,然后运行 [`deflate`](https://github.com/raphael2025/deflate) 的 `placebo`、`deflated_sharpe`(惩罚回看窗口网格)、`pbo`。缓慢版本看起来像趋势跟随且能幸存;快速版本则死于成本和安慰剂检验。
