# 02 · 图表形态

交易者从价格图表中读出的视觉形态——在诚实、因果对齐的时间戳、计入成本并与匹配安慰剂对照的条件下接受检验。

| 形态 | 判决 | 原因 |
|---|---|---|
| [头肩形态](head-and-shoulders.md) | ❌ | 未来函数:枢轴点和颈线只有事后才能知道;一旦因果对齐打时间戳,突破后收益与安慰剂无法区分,计入成本后持平甚至为负。 |
| [K线形态](candlestick-patterns.md) | ❌ | 多重检验 + 成本:数十种可调参的命名形态保证了样本内总能挑出赢家;DSR 把整个家族压缩到约等于零,且扣除费用后并不能跑赢安慰剂。 |

图表形态正是 **未来函数破坏力最强** 的家族——事后标记形态会虚高"胜率",而一旦把摆动点按诚实的确认时刻打上时间戳、并与匹配安慰剂入场对照,这种胜率便荡然无存。
