# K线形态

> 一句话:被命名的单根、两根、三根K线形状(十字星、锤子线、吞没、晨星/暮星……),据称可预测短期反转或延续。**判决:❌ 已证伪**(作为独立信号)。

## 它是什么

K线形态把一根K线的开盘价、最高价、最低价、收盘价之间的关系编码成被命名的形状:

- **十字星(Doji)** —— 开盘价 ≈ 收盘价(实体极小):"犹豫不决"。
- **锤子线 / 流星线** —— 实体很小、带长下影/上影:动作被拒绝。
- **看涨/看跌吞没** —— 实体完全吞没前一根实体:"动能反转"。
- **晨星 / 暮星** —— 三根K线构成的反转序列。

这套框架源自日本大米交易者,经史蒂夫·尼森(Steve Nison)在西方推广开来。命名形态多达数十种,常常配合"确认"规则(例如随后一根的跟进收盘)。

## 主张

这些特定形状携带短期预测信息——比如,低点处的一根锤子线可靠地预示反弹,一组暮星预示下跌。

## 检验

每种形态都被编码成加密永续合约(BTC/ETH/SOL)和股票上的 **机械信号**。K线确实有一点做对了:一根K线只有在收盘时才算完成,因此在K线收盘处发出信号在因果上是干净的——**这里没有未来函数**(不像[头肩形态](head-and-shoulders.md),那里事后标记枢轴点才是致命问题)。真正的麻烦在于 **家族的规模**:数十种命名形态 × 可调参的确认规则 = 一个巨大的搜索面。我们把整个集合当作一次知晓筛选的实验来处理,**内化成本**,并将入场与 **匹配安慰剂** 对照(按时段和动量匹配的随机入场)。

发现:

- **样本内总有东西看起来不错。** 形态和确认变体数量如此之多,单凭运气就能保证样本内出现赢家。
- **DSR 把整个家族压垮。** 一旦 **Deflated Sharpe Ratio** 惩罚尝试过的形态/规则数量,样本内的明星便向零收缩;PBO 很高。
- **计入成本后相对安慰剂无优势。** 扣除费用后,这些形态触发的收益与匹配安慰剂入场在统计上无法区分。预测信息约等于零。

## 判决:❌ 已证伪

这里的致命因素是 **多重检验 + 成本 + 相对安慰剂无优势**——而非未来函数。作为独立的反转/延续信号,K线形态无法存活:为家族规模做收缩、扣净成本后,优势消失。这与学术界的结论一致:一旦计入数据窥探,K线策略并不增加价值。

一个公允的说明:有些形态只是 *描述性* 的——一根十字星确实描述了收盘回到开盘附近的一段行情,一根长下影确实描述了盘中被拒绝。这是叙述图表的合理词汇,但 *不是* 能改变判决的优势。我们就其主要用途——作为可交易信号——把该主题标记为 ❌,同时承认其描述性角色无害。

## 自己试试

把每种形态编码成收盘确认的信号,把完整的形态清单当作一个搜索家族,扣净真实成本,然后运行 [`deflate`](https://github.com/raphael2025/deflate):`placebo`、`deflated_sharpe`(这是承重检验——它必须惩罚 *整个* 家族,而不是单个形态)以及 `pbo`。DSR 这一步正是家族死亡之处。
