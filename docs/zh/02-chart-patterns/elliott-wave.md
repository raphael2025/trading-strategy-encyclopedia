# 艾略特波浪理论(Elliott Wave Theory)

> 一句话:声称市场以可重复的 5 浪推动 + 3 浪调整的分形结构运动,且可提前数浪。**判决:❌(主观、不可证伪、含未来函数)。**

## 它是什么

拉尔夫·尼尔森·艾略特(Ralph Nelson Elliott,1930 年代)提出,群体心理驱动价格以自相似的分形"波浪"运动。一个完整循环是**沿趋势方向的 5 浪(一个推动浪:1-2-3-4-5),随后是 3 浪调整(A-B-C)。**每一浪又细分为形态相同的更小波浪,贯穿所有时间周期。一套规则(第 3 浪绝不是最短的;第 4 浪不与第 1 浪重叠;等等)以及各浪长度之间的斐波那契关系被用来"数"出当前所处的位置,并预测下一步走势。

## 主张

主张是:如果你正确识别出自己处在波浪计数中的哪个位置,你就知道下一步走势是推动浪(顺势交易)还是调整浪(逆势或观望),而斐波那契比率会给出目标位。支持者将其呈现为一套完整的预测框架。

## 检验

艾略特波浪是一个*无法被诚实地送入闸道*的理论的典型范例,而这本身就是判决:

- **不可证伪 / 无限灵活。**这套规则允许如此之多的替代数浪方式(以及"延长浪""失败浪""楔形浪")以致几乎任何价格路径都能在事后被重新标注以拟合。一个能解释一切结果的理论什么也预测不了。这与[我们的方法论](../../../METHODOLOGY.md)所要求的恰恰相反——一个*因果的、固定的*规则。
- **构造上即含未来函数。**"我们当时处在第 3 浪"是一个只有在第 3 浪*完成之后*你才能有把握说出的陈述。实时数浪常常被反复修正,这意味着所谓"信号"是一段事后叙事,而非可交易的触发条件——这与[威科夫(Wyckoff)](../07-market-mechanics/wyckoff-accumulation-distribution.md)和[谐波形态](harmonic-patterns.md)的缺陷如出一辙。
- **没有可复现的边际。**因为两位分析师在同一张图上会得出不同的数浪,所以没有客观规则可供回测,没有[安慰剂检验](../../../METHODOLOGY.md)对照,也没有 Deflated-Sharpe 数值。数十年的实践从未产出任何大样本、经同行评审的样本外、扣成本后仍存活的边际证据。

## 判决:❌

艾略特波浪在还没走到任何一次成本计算之前就已失败:它主观、可被无穷修订,且只在事后"奏效"。其中那一点真理内核——市场会趋势也会调整,持续性是真实存在的——被机械的[趋势跟随](../03-strategies/trend-following.md)和[横截面动量](../03-strategies/cross-sectional-momentum.md)以诚实得多的方式捕捉,它们客观、可复现,并能通过闸道。如果你无法把规则写下来、使两个人得出相同的交易,你就无法检验它;而如果你无法检验它,它就维持 ❌。

## 自己试试

这个练习*本身*就是教训:试着把一次艾略特数浪编码为仅依赖历史价格的确定性函数。你会发现,每一个有意思的决策(哪段摆动是第 1 浪的起点、这是第 4 浪还是一个新的推动浪)都需要未来信息或一次自由裁量的判断。正是这种不可消除的主观性使它无法通过 [`deflate`](https://github.com/raphael2025/deflate)——根本没有任何固定的东西可供 deflate。
