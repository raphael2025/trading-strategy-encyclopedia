# 08 · 加密专属

加密永续市场独有的结构性特征——用与其他一切相同的诚实方式来检验,把那一个真正的优势与你只需理解的机制区分开来。

| 主题 | 判决 | 原因 |
|---|---|---|
| [资金费率 carry](funding-rate-carry.md) | ✅ | 来自一笔真实的、交易所支付的现金流的近乎无风险 carry——Sharpe 非常高,但有一条真实的左尾,所以仓位 ≤20%。加密中最干净、最诚实的优势。 |
| [永续与清算](perpetuals-and-liquidations.md) | ⚠️ | 机制参考,不是策略——标记价格、保证金、清算级联、ADL。解释了为什么 carry 有效、为什么杠杆是致命的。 |

加密的结构性怪癖——**永续资金费**——也是它唯一一个最干净的已验证优势;这里其他的一切都是你需要理解的机制,好让你不被清算。
