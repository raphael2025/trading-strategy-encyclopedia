# 05 · 交易心理

市场的行为面——认知偏差、情绪与情绪指标——都要拿来检验那个唯一重要的问题:它们当中有任何东西能*预测价格*吗?还是只是帮你避免自我破坏?

| 主题 | 判决 | 原因 |
|---|---|---|
| [认知偏差](cognitive-biases.md) | ⚠️ | 不是信号——没有任何东西可供回测;它对纪律和风险确有用处,但不能预测。 |
| [恐惧贪婪指数](fear-and-greed-index.md) | ❌ | 滞后且基本是循环论证(贪婪*就是*近期收益);阈值可调,作为信号无法打赢安慰剂检验。 |

诚实的框架:**心理学是风险管理,不是 alpha。** 了解自己的偏差能让你不去破坏自己已有的真实优势——例如,不去过早砍掉[趋势跟随](../03-strategies/trend-following.md)所依赖的那些罕见的大赢家——但它并不能预测价格。把它当作防御工具来用;不要误以为它是信号。
