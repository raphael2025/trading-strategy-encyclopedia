# 交易中的认知偏差

> 一句话:那一整套被系统记录的心理错误——损失厌恶、近因效应、过度自信、FOMO——交易者被告知要"掌控"它们才能取胜。**判决:⚠️ 不是信号**(对纪律有用,但不能预测)。

## 它是什么

一组在市场中反复出现、有充分文献记录的决策错误:

- **损失厌恶 / 处置效应**——过早卖掉赢家,过久持有输家。
- **近因偏差**——过度看重最近的几根 K 线并将其外推。
- **确认偏差**——只去寻找支持你已有仓位的证据。
- **过度自信**——在一波好运之后过度交易、过度放大仓位。
- **锚定**——死盯着某个入场价或某个整数关口。
- **FOMO**——追逐已经发生的行情。
- **赌徒谬误**——认为反转"该来了"而预期它发生。
- **沉没成本**——为了证明最初的决策正确而向输家加仓。

## 主张

大众交易话术很诱人:"掌控你的心理,你就会赢。"它把情绪控制当成那块缺失的优势——仿佛只要消除你的偏差,本身就能产生一条收益流。

## 检验 / 现实

这里没有任何东西可供回测。**了解确认偏差并不是一个信号**——它不生成入场点,没有 RankIC,无法做选择感知(selection-aware)的打分,因为它对价格不作任何预测。诚实地重述这一主张就会让它自行瓦解:心理学不是 alpha。

但它在另一个方向上确实有用——作为**风险与流程**,而非预测:

- 处置效应恰恰就是那种**摧毁[趋势跟随](../03-strategies/trend-following.md)盈亏比**的行为:过早砍掉那些罕见的、约 6.5 盈亏比的赢家(一个约 36% 胜率的策略),就等于截断了那条供养一切的肥尾。这个偏差攻击的正是优势所依赖的那个机制。
- 过度自信与 FOMO 表现为过度放大仓位——靠基于规则的[仓位与波动目标](../06-risk-management/position-sizing-and-vol-targeting.md)来中和,而不是靠意志力。
- 对它们所有人的防御都是**预先承诺**:成文规则、机械化的[止损](../06-risk-management/stop-losses.md),以及在你冷静时、交易之前就定好的固定仓位。

## 判决:⚠️ 不是信号(对纪律有用,但不能预测)

偏差意识的价值是**防御性的**:它让你不去破坏自己已有的优势。它不具备**预测性**:它永远不会告诉你该买什么。要诚实地分清你在用哪一半——把两者混为一谈本身就是一种偏差。

## 自己动手试试

你无法回测一个偏差,但你可以回测它的*代价*。取一条已验证的趋势跟随收益流,然后叠加一个处置效应来重新模拟它——对赢家过早止盈,让输家越过止损继续持有——看着 Sharpe 随着盈亏比被侵蚀而崩塌。把两者都跑一遍 [`deflate`](https://github.com/raphael2025/deflate)。即便偏差本身不是信号,这条行为漏洞也是可以度量的。
