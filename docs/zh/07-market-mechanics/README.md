# 07 · 市场机制

大玩家如何真正地推动价格——以及一个机制是**真实存在**与**可交易**之间那道至关重要的区别。这些现象确实会发生;诚实的结论是,对零售交易者而言,"发现它并据此择时入场"无法在未来函数、安慰剂检验和成本面前存活。它们的价值在于**防御**(止损*不要*放在哪里、为何要保持低杠杆),而非预测。

| 主题 | 判决 | 原因 |
|---|---|---|
| [流动性猎杀](liquidity-hunts.md) | ⚠️ 真实 / ❌ 作为择时 | 清算级联和订单池是真实的;扫单逆势入场打不过安慰剂,且在日内成本上失血。事后标签。 |
| [止损猎杀](stop-hunting.md) | ⚠️ 真实 / ❌ 作为择时 | 止损确实会被触发——可用于防御——但"交易这场猎杀"就是那个已被证伪的流动性扫单逆势交易。 |
| [市场操纵](market-manipulation.md) | ⚠️ 真实 / ❌ 作为信号 | 虚假挂单/对敲/拉高出货/逼仓都有据可查且违法;能盈利的侦测靠的是 HFT 延迟或事后取证,而非零售边际。 |
| [威科夫吸筹与派发](wyckoff-accumulation-distribution.md) | ❌ 作为择时 / ⚠️ 作为视角 | 各阶段只能事后知晓;主观标注会过拟合;入场价位是拥挤的流动性。 |

统一的教训:**机制 ≠ 边际。**懂得清算级联或虚假挂单墙如何运作,会让你成为更好的*防御者*(在清算簇附近减小仓位、不要把止损放在显而易见的价位、保持低杠杆)。它并不会递给你一个交易布局——我们检验过的每一条零售时间周期"交易这场操纵"的规则,都死于事后标签、拥挤和换手成本。
