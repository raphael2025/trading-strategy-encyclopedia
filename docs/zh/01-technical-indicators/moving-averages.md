# 移动平均线与均线交叉

> 一句话:价格的平滑平均(SMA/EMA),其交叉点——"黄金交叉"与"死亡交叉"——被当作买/卖信号兜售。**判决:❌ 已证伪**(作为独立的、调过参的交叉信号)。

## 它是什么

移动平均线在一个回看窗口上平滑价格。两种常见形式:

```
SMA(N) = 最近 N 根收盘价的简单均值(等权重)
EMA(N) = 指数加权均值(越近的收盘价权重越大)
```

交叉规则把一条**快**均线和一条**慢**均线配对。当快均线*上穿*慢均线时是**黄金交叉**;当它*下穿*时是**死亡交叉**。被引用最多的一对是 50/200。

## 它的主张

移动平均线滤掉噪声、揭示底层趋势,所以当快均线超越慢均线时动量翻多(黄金交叉=买入),当它跌破时翻空(死亡交叉=卖出)。尤其 50/200 交叉被当作整个市场的状态切换。

## 检验

我们把均线交叉当作一条**独立的、机械式的入场规则**,在加密永续合约(BTC/ETH/SOL)和股票上做了评估,信号在 bar 收盘时打时间戳(无未来函数),并将**吃单/taker 费用 + 滑点 + 资金费内化**。(fast, slow) 长度被当作一个搜索网格处理,以便诚实地度量选择偏差。

结论,与索引一致:

- **扣成本后无样本外优势。** 样本内"最优"的长度对无法延续到样本外切分——正是在[判决索引](../00-verdict-index.md)里**五种独立证伪方法**下统统失败的那类独立指标。
- **对选择有意识便死。** 搜索长度对会抬高样本内 Sharpe,但 **Deflated Sharpe/DSR**(为试过多少对而惩罚)向零崩溃,且 PBO 很高——样本内最优的那一对不过是样本外的噪声。
- **成本主导一切。** 交叉在区间震荡市场里反复打脸;在更快的那些参数对上,光是换手成本就是拖垮中频方向预测的同一种拖累(≈ **每年 −86% 进了费用**)。
- **打不过安慰剂检验(placebo test)。** 交叉入场与按时段和当下动量匹配过的随机入场无从区分。

## 判决:❌ 已证伪

作为一个独立的、调过参的交叉信号,均线交叉过不了诚实检验。致命之处是**多重检验**(用哪一对?50/200 是民间俗成挑出来的,不是推导出来的——而样本内最优的那一对无法泛化)外加**成本**(在震荡中反复打脸)。"黄金交叉就是买入"是对选择视而不见的民间传说。

**关键的微妙之处——指标本身没错,错的是框定方式。** 移动平均线是一个经过验证的[趋势跟随](../03-strategies/trend-following.md)系统里完全够用的*构件*。差别全在*如何*使用它:经过验证的趋势跟随是**缓慢的**(骑跨数月的走势,而不是在震荡里追交叉)、**对成本有意识的**(换手预算被管控)、**对选择有意识的**(参数搜索被惩罚,而非精挑细选),并以不对称的盈亏比让赢家奔跑(Sharpe ~1.1,胜率 ~36%,盈亏比 ~6.5)。同样的原始原料(一条平滑价格)活在两者之中——只有独立调参信号这种框定方式在这里死掉。

## 自己动手试

在 bar 收盘上生成黄金交叉做多 / 死亡交叉做空的入场信号,每次换手扣掉现实的吃单/taker 成本,然后把收益序列跑过 [`deflate`](https://github.com/raphael2025/deflate):检查 `placebo`(对比匹配过的随机入场)、`deflated_sharpe`(惩罚你搜索过的 (fast, slow) 网格)和 `pbo`(样本内最优的那一对能在样本外幸存吗?)。失败的正是 DSR/PBO 这一步。
