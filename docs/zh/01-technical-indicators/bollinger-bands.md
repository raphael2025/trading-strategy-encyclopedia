# 布林带

> 一句话:一条移动平均线被一对按波动率缩放的带子包裹,既可读作"淡化边缘"的回复信号,也可读作"挤压突破"信号。**判决:❌ 已证伪**(作为独立的回复信号)。

## 它是什么

布林带由 John Bollinger 在 1980 年代提出,在一条移动平均线上画出一个由近期波动率设定的包络:

```
middle band = SMA(20)
upper band  = SMA(20) + 2 × stdev(20)
lower band  = SMA(20) − 2 × stdev(20)
%B          = (price − lower) / (upper − lower)      (价格在带内的位置)
bandwidth   = (upper − lower) / middle               (带子有多宽)
```

两种相互矛盾的民间读法:**(a) 均值回复**——价格触及上轨是"超买,卖出";触及下轨是"超卖,买入"(淡化边缘)。**(b) 挤压突破**——当 bandwidth 收缩到多周期的低点时,一波大的方向性走势正在"蓄势",于是去交易突破*出*带子的那一刻。

## 它的主张

±2 个标准差"包含"了"大部分"价格行为,所以触及外轨是一次被统计上拉伸了的走势,理应弹回(回复);或者反过来,狭窄的带子(低波动)预示一次扩张,于是从挤压中突破是可交易的。

## 检验

我们把**两种**读法都当作独立的机械规则,在加密永续合约(BTC/ETH/SOL)和股票上做了评估,信号在 bar 收盘时给出(无未来函数),并将**吃单/taker 费用 + 滑点 + 资金费内化**。(period, num_std) 参数被当作一个搜索网格处理。

结论,与索引一致:

- **扣成本后无样本外优势。** 样本内"最优"的 (period, num_std) 无法延续到样本外——这正是在[判决索引](../00-verdict-index.md)里**五种独立证伪方法**下统统失败的那类独立指标。
- **对选择有意识便死。** 搜索带子参数会抬高样本内 Sharpe,但 **Deflated Sharpe/DSR** 向零崩溃,且 PBO 很高。两种相互矛盾的解读让情况更糟:你*总能*在样本内套上其中一个,而这恰恰是 DSR/PBO 专门要惩罚的那种灵活性。
- **成本主导一切。** 触带和突破触发得足够频繁,光是换手成本就构成了同样的拖累(≈ **每年 −86% 进了费用**),足以拖垮更高频的策略。
- **打不过安慰剂检验(placebo test)。** 触带入场与按时段和当下动量匹配过的随机入场无从区分。

## 判决:❌ 已证伪

作为一个独立的回复信号,布林带过不了诚实检验。致命之处是**多重检验**(20/2 默认值是民间调出来的,*而且*拥有两种相反的解读保证了在任何样本内窗口里总有一个能套上)外加**成本**。淡化触带和拖垮[均值回复](../03-strategies/mean-reversion.md)的接飞刀问题是同一个:在真实趋势里,价格能贴着上轨走好几周,而"触带就卖"把亏损的那一边塞给你。

突破读法 (b) 是另一个问题——挤压→突破在[突破](../03-strategies/breakout.md)下被列为 🔶 **有条件**,它只有在一个对成本有意识、对选择有意识的框架内才幸存,而不是作为一条独立调过参的带子规则。

## 自己动手试

在 bar 收盘上生成淡化带子(下轨做多 / 上轨做空)的入场信号,每次换手扣掉现实的吃单/taker 成本,然后把收益序列跑过 [`deflate`](https://github.com/raphael2025/deflate):检查 `placebo`(对比匹配过的随机入场)、`deflated_sharpe`(惩罚 (period, num_std) 网格*以及*两种解读之间的选择)和 `pbo`(样本内最优配置能在样本外幸存吗?)。失败的正是 DSR/PBO 这一步。
