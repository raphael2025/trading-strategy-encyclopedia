# 06 · 风险管理

如何诚实地控制风险——用与策略相同的方式来检验:因果时间戳、内化成本,以及对*感觉*安全与*实际*安全之间区别的敏锐眼光。

| 主题 | 判决 | 原因 |
|---|---|---|
| [仓位与波动目标](position-sizing-and-vol-targeting.md) | ✅ | 回撤是一个近乎线性的旋钮——仓位减半,回撤减半,而 Sharpe/Calmar 大致不变。这是一个关于风险如何缩放的恒等式;没有什么可过拟合的。 |
| [止损](stop-losses.md) | ❌ / 🔶 | 紧止损会杀死优势所依赖的盈亏比;宽 ATR 移动止损作为灾难保险有帮助。判决依宽度而分裂。 |
| [杠杆](leverage.md) | ❌ | 是破产开关,不是乘数——几何拖累加上吸收态的清算,意味着高杠杆下的期望增长为负。保持 ≤1×。 |

风险管理是散户真正能改变局面的唯一地方——但杠杆点是**仓位大小**,不是巧妙的出场,也不是杠杆。控制你下注的*规模*,让已验证的优势自由呼吸。
