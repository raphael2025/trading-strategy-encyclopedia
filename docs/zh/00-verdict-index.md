# 判决索引

所有策略/指标一览。标签:✅ 已验证 · ❌ 已证伪 · 🔶 有条件 · ⚠️ 未测。
**资产范围:** 加密永续(BTC/ETH/SOL)与美股,散户可执行,含成本。见 [方法论 METHODOLOGY](../../METHODOLOGY.md)。

> 这样去读它:*"一旦你不再自欺,这个边还能不能活下来?"* 对大多数出名的东西,答案是不能——而这件事本身就值得知道,因为它能让你别再为江湖传说白白烧手续费。

## 均值回复、SMC / ICT

| 策略 | 判决 | 证据 |
|---|---|---|
| 趋势中的均值回复 / "抄底" | ❌ | 在加密的趋势行情里 = 接飞刀。 |
| 趋势中做市 / 网格 | ❌ | 比空仓更差;趋势把你碾过去。 |
| 供需区(SMC) | ❌ | 相对匹配安慰剂入场无边。 |
| Fair Value Gap(FVG)回踩 | ❌ | 事件研究,跨 3 个币 × 多个持有期共 36 次检验:过不了多重检验 + 样本外。 |
| 流动性扫荡反转("止损猎杀"反向) | ❌ | 触发后的收益计入成本为净负;相对安慰剂无边。 |

## 趋势、动量、carry(幸存者)

| 策略 | 判决 | 证据 |
|---|---|---|
| 趋势跟随(双均线,让利润奔跑) | ✅ | Sharpe ~1.1;胜率约 36% 但盈亏比约 6.5——边在于*不*砍掉赢家。 |
| 横截面动量 + 非流动性(流动性宇宙) | ✅ | 多因子,头/尾各 20%;Sharpe ~1.2。 |
| 资金费 carry(现货-期货套利:多现货 / 空永续) | ✅ | 近乎无风险的资金费收割;Sharpe 很高,但有真实的尾部风险——仓位 ≤20%。**加密里最干净、最诚实的边。** |
| 组合(趋势 + 横截面 + carry),相互相关性低 | ✅ | Sharpe ~1.7,**DSR 0.93**,回撤约 −22%——风险调整后的最佳形态。 |
| 等权买入持有(优质股票) | ✅ | Sharpe ~1.11——一个极难打败的基准。复杂方案必须先证明能打赢*它*。 |

## 风险管理与执行

| 主题 | 判决 | 证据 |
|---|---|---|
| 用仓位 / 波动目标控制回撤 | ✅ | 回撤几乎是一个线性旋钮:仓位减半 → 回撤减半,Sharpe/Calmar 近乎不变。**这才是控制风险的方式**——不是花哨的择时滤波。 |
| 宽 ATR 移动止损 | 🔶 | 有帮助:在保留趋势跟随所需的肥右尾的同时,砍掉灾难性反转。 |
| 紧止损 / 早止盈 / 阻力位离场 | ❌ | 砍掉那几个大赢家 → 摧毁整个边赖以生存的盈亏比。 |
| 把杠杆当"边的放大器" | ❌ | 那是破产开关,不是乘数。10% 资金 × 50× → 一根普通插针就归零。保持 ≤1×。 |
| 高周期信号后用低周期"更好入场"择时 | ❌ | 增加来回打脸 + 错失行情;还不如直接照信号动手。 |

### 元教训

> 跨越一次穷尽式搜索,**交易者所信的大部分都是经不起诚实检验的江湖传说。** 活下来的都很无聊:跟随趋势、收割 carry、分散不相关的边、控制仓位,并尊重一个简单的买入持有就是凶狠的对手。**克制胜过努力。** 这个索引的价值不是一个策略——而是*别在那些已死的东西上浪费数年(和手续费)。*
