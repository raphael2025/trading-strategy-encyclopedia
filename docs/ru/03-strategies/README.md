# 03 · Стратегии

Целостные торговые *стратегии* — логика входа, правила удержания и дисциплина выхода — протестированные с честной причинной разметкой времени, с внутренним учётом издержек и с дефляцией с поправкой на отбор (DSR/PBO), а не просто отдельные индикаторы.

| Стратегия | Вердикт | Почему |
|---|---|---|
| [Трендследование](trend-following.md) | ✅ | Едет на устойчивости в том виде, в каком моментум реально существует; достаточно медленное, чтобы перекрыть издержки, с поправкой на отбор. Sharpe ~1.1, доля прибыльных ~36%, но коэффициент выплаты ~6.5 — преимущество в том, чтобы *не* срезать прибыльные сделки. |
| [Кросс-секционный моментум](cross-sectional-momentum.md) | ✅ | Относительный, диверсифицированный, низкая корреляция; Sharpe ~1.2 и проходит DSR/PBO — но только на ликвидной, но не перенасыщенной вселенной (мегакапы DSR ≈ 0.004). |
| [Пробой](breakout.md) | 🔶 | Реален только как медленный пробой канала с учётом издержек (по сути трендследование); наивный внутридневной «покупай пробой» умирает от ложных пробоев + издержек на оборот. |
| [Возврат к среднему](mean-reversion.md) | ❌ | В трендовом режиме крипты фейд экстремумов = ловля падающего ножа; нетто-отрицательно после издержек, нет преимущества против плацебо. (Легитимен только в рыночно-нейтральном парном трейдинге / стат-арбе.) |
| [Копитрейдинг](copy-trading.md) | ❌ | RankIC ≈ 0, ошибка выжившего, и вы исполняетесь позади лидера как его выходная ликвидность — рейтинг с нулевой прогнозной информацией не является стратегией. |
| [Покупай и держи (равновзвешенный)](buy-and-hold.md) | ✅ | Sharpe ~1.11 на качественных акциях, почти нулевые издержки, нечего переобучать — бенчмарк, который должна побить любая активная стратегия (большинство не может). |
| [Парный трейдинг / стат-арб](pairs-trading.md) | 🔶 | Реальное преимущество относительной стоимости, но затухшее с ~2002 года; в крипте коинтеграция нестабильна, факторные шоки бьют по обеим ногам, а поиск пар переобучается. |
| [Сетка](grid-trading.md) | ❌ | Прибыль в диапазоне, затем тренд вас переезжает, пока вы накапливаете убыточную позицию; издержки на каждой ступени. Только настоящий маркет-мейкинг с управлением запасами — 🔶. |
| [DCA (усреднение стоимости)](dca-dollar-cost-averaging.md) | ⚠️/🔶 | Инвесторский DCA — инструмент денежного потока/поведения (по матожиданию проигрывает единовременному входу, не альфа); кредитные боты «усреднения вниз» проваливаются как сетки. |
| [Факторы на мегакапитализациях](mega-cap-factors.md) | ❌ | DSR ≈ 0.004 — факторная премия живёт в малой/средней капитализации; мегакапы слишком эффективны, а нескольких имён недостаточно для кросс-секционной широты. |

Все выжившие делают одно и то же: едут на **моментуме/устойчивости** медленно, с учётом издержек и с поправкой на отбор, и диверсифицируют в **низкокоррелированные комбинации** (тренд + кросс-секция + carry достигает Sharpe ~1.7). Мёртвые либо **борются с трендом** (возврат к среднему), либо **гонятся за миражами** (копитрейдинг = быть чьей-то выходной ликвидностью).
