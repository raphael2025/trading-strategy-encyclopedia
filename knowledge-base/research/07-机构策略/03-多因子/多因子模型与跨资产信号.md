## 一、多因子模型基础

### 1. 概念

**多因子模型（Multi-Factor Model）**：
- 资产收益 = 多个共同因子的线性组合
- 每个因子捕捉一种风险溢价
- 用于解释、预测、风险管理

**与单因子模型的区别**：
- 单因子：CAPM（市场）
- 多因子：Fama-French（市场+规模+价值）、Carhart（+动量）等

### 2. 因子投资的历史

**1960s-1970s**：
- CAPM（Sharpe 1964）：E(R) = Rf + β(E(Rm) - Rf)
- 单因子时代

**1980s-1990s**：
- Fama-French 三因子（1992）：+ 规模（SMB）+ 价值（HML）
- Carhart 四因子（1997）：+ 动量（MOM）

**2000s-2010s**：
- Fama-French 五因子（2015）：+ 盈利（RMW）+ 投资（CMA）
- AQR 推动"Smart Beta" 浪潮
- 因子 ETF 爆发

**2010s-2020s**：
- 加密因子研究
- 链上因子
- 另类数据因子
- 机器学习因子

### 3. 加密资产因子的特殊性

| 维度 | 传统市场 | 加密市场 |
|------|---------|---------|
| 因子数量 | 已成熟 5-10 个 | 探索中 20+ 个 |
| 数据可得 | 高频 tick + 财报 | 24/7 行情 + 链上 |
| 因子稳定性 | 长期稳定 | 高频变化 |
| 资产数量 | 4000+ 股票 | 2,000,000+ 代币（但实际可交易 5,000+）|
| 因子研究 | 30+ 年 | 5-10 年 |

## 二、加密市场核心因子

### 1. 动量因子（Momentum）

**定义**：过去 N 天收益率高的币种，未来继续表现好

**经典研究**：
- Jegadeesh & Titman (1993)：动量效应
- 股票 3-12 月动量，年化超额 8-12%
- AQR 实测：长期存在

**加密应用**：
- 过去 30 天收益率 → 排序
- Top quintile - Bottom quintile = 动量因子收益
- 加密市场动量更强（散户主导）

**Backtest 结果**（BTC + Top 20 山寨币）：
- 2018-2024 年化 30-50%
- Sharpe 1.0-1.5
- Max DD 30-40%

**改进**：
- 残差动量（Idzorek & Xiong 2014）
- 动量崩溃保护（Daniel & Moskowitz 2016）

### 2. 价值因子（Value）

**定义**：估值低的资产未来表现好

**传统指标**：
- P/E（市盈率）
- P/B（市净率）
- EV/EBITDA

**加密对应**：
- **MVRV**：Market Cap / Realized Cap
- **NVT**：Network Value / Transactions
- **NVM**：Network Value / Metcalfe（活跃地址²）
- **BTC Mayer Multiple**：Price / 200D MA

**典型规则**：
- MVRV < 1：深度低估 → 买入信号
- MVRV > 3.5：高度高估 → 卖出信号
- 历史回测：BTC MVRV 策略年化 30%+

**Glassnode 数据显示**：
- 2015-2024 MVRV < 1 时间占 20%
- 这些时间点买入并持有 1 年，平均收益 100-200%
- MVRV > 3.5 时间占 15%，这些时间点卖出信号

### 3. 规模因子（Size）

**传统**：小盘股 > 大盘股（规模溢价）

**加密对应**：
- 小市值代币 > 大市值
- 但**生存者偏差**很大
- 很多小币归零

**实战**：
- 排除市值 < 1 亿美元的币
- 排除 30 天新上线的币
- 排除日交易量 < 100 万美元的币

**回测**（2020-2024）：
- 小市值组合年化 80%（但 Max DD 70%）
- 中市值组合年化 40%（Max DD 40%）
- 大市值组合年化 30%（Max DD 30%）
- **风险调整**：中市值最优

### 4. 波动率因子（Volatility）

**传统发现**：
- 低波动率股票 > 高波动率股票（"低波动率异常"）
- Baker, Bradley, & Wurgler (2011)

**加密对应**：
- 过去 30 天波动率
- 低波动率币种：BTC, ETH, 大型稳定币
- 高波动率币种：meme、DeFi 新币

**反直觉**：
- 加密市场**高波动率**币种反而收益高
- 散户偏爱"刺激"
- 但风险调整后低波动率更优

### 5. 链上因子（On-Chain）

**核心指标**：

**Exchange Netflow**：
- 净流入（> 0）：准备卖出 → 看空
- 净流出（< 0）：提币囤积 → 看多
- **历史回测**：3 天累计净流出 > 1000 BTC，看多 7 天

**MVRV**：
- 见价值因子

**SOPR（Spent Output Profit Ratio）**：
- 已花费 BTC 实现价值 / 原始价值
- SOPR > 1：获利回吐
- SOPR < 1：亏损持有
- **信号**：SOPR 长期 < 1 转为 > 1，反弹信号

**NVT（Network Value to Transactions）**：
- 类似 P/E
- NVT > 100：网络价值高估
- NVT < 50：网络价值低估
- **数据**：Woobull, Coinmetrics

**Active Addresses**：
- 活跃地址数
- 长期上升 → 网络效应↑
- 突然下降 → 风险

**Hashrate**（仅 BTC）：
- 矿工信心
- 长期上升 → 健康
- 急剧下降 → 风险

**Long-Term Holder SOPR**：
- 长期持有者（> 155 天）的 SOPR
- 长期持有者卖出 → 顶部信号
- 长期持有者持有 → 累积信号

### 6. 资金流因子（Funding Rate）

**Funding Rate 因子**：
- Funding > 0.05%/8h：多仓拥挤 → 做空信号
- Funding < -0.05%/8h：空仓拥挤 → 做多信号

**实战**：
- Funding Rate 与价格反转相关
- **Glassnode 2024 研究**：Funding Rate 因子年化 15-25%

### 7. 情绪因子（Sentiment）

**指标**：
- 恐惧贪婪指数
- 社交媒体活跃度
- 谷歌趋势
- 期权 IV

**典型**：
- 极度贪婪（> 90）→ 看空
- 极度恐惧（< 10）→ 看多
- 历史上有效，但需要结合其他信号

### 8. 流动性因子（Liquidity）

**指标**：
- 24h 交易量
- 订单簿深度
- Bid-Ask 价差
- Amihud 非流动性指标

**实战**：
- 流动性低的币种流动性溢价
- 但实际可能是流动性陷阱
- 排除流动性 < 100 万美元的币

## 三、因子合成方法

### 1. 简单加权

**等权**：
- 每个因子权重 1/N
- 简单但忽略了因子质量

**IC 加权**：
- 权重 ∝ |IC 因子|
- IC 高的因子权重高

**信息比率加权**：
- 权重 ∝ IR 因子 = IC × √(N)
- 理论最优

### 2. 回归合成

**Fama-MacBeth 回归**（1973）：
- 每月对所有股票回归
- 用回归系数作为因子权重
- **问题**：估计误差

**Shrinkage 估计**：
- 用 James-Stein 收缩
- 缓解估计误差

**LASSO 回归**：
- 自动选择因子
- L1 正则化
- 适合因子数量大的情况

### 3. 机器学习

**树模型**：
- XGBoost、LightGBM
- 特征重要性 = 因子权重
- **问题**：黑箱、不稳定

**神经网络**：
- 多层感知机
- CNN（捕捉时序模式）
- Transformer（最新趋势）
- **问题**：过拟合

**特征工程**：
- 因子标准化
- 因子交互
- 因子时间衰减

### 4. 最优组合

**最大化 IR**：
- Min 波动率，目标收益
- 求解最优因子权重

**最大化 Sharpe**：
- 假设因子收益正态
- 求解最优组合

**风险平价因子**：
- 每个因子贡献等量风险
- 适合多因子等贡献

## 四、加密因子模型实战

### 1. 因子库

**定义因子**（Python 示例）：
```python
def compute_factors(df):
    factors = pd.DataFrame(index=df.index)
    
    # 1. 动量（过去 30 天）
    factors['mom_30d'] = df['close'].pct_change(30)
    
    # 2. 动量（过去 7 天）
    factors['mom_7d'] = df['close'].pct_change(7)
    
    # 3. 价值（MVRV）
    factors['mvrv'] = df['close'] / df['realized_cap']
    
    # 4. 波动率（过去 30 天）
    factors['vol_30d'] = df['close'].pct_change().rolling(30).std()
    
    # 5. 资金流（Funding Rate）
    factors['funding'] = df['funding_rate']
    
    # 6. 链上（活跃地址）
    factors['active_addr'] = df['active_addresses'].pct_change(30)
    
    # 7. 情绪（恐惧贪婪）
    factors['fear_greed'] = df['fear_greed_index']
    
    # 8. 流动性（交易量）
    factors['volume_ratio'] = df['volume'] / df['volume'].rolling(30).mean()
    
    return factors
```

### 2. 因子打分

**Z-Score 标准化**：
```python
factors_z = (factors - factors.mean()) / factors.std()
```

**Rank 标准化**（更稳健）：
```python
factors_rank = factors.rank(pct=True)
```

### 3. 综合得分

**加权**：
```python
# 等权
composite = factors_z.mean(axis=1)

# IC 加权
weights = {'mom_30d': 0.25, 'mvrv': 0.20, 'vol_30d': 0.15, 
           'funding': 0.15, 'active_addr': 0.10, 'fear_greed': 0.10,
           'volume_ratio': 0.05}
composite = (factors_z * pd.Series(weights)).sum(axis=1)
```

### 4. 投资组合

**Top N + Bottom N**：
```python
# 月度再平衡
top_n = 5
bottom_n = 5

long_portfolio = composite.nlargest(top_n).index
short_portfolio = composite.nsmallest(bottom_n).index
```

### 5. 回测

**关键指标**：
- IC（信息系数）：因子与未来收益的相关性
- Rank IC：秩相关
- Turnover（换手率）
- Long-Short 收益
- Long-Short Sharpe

**典型回测结果**（多因子综合）：

| 因子 | IC | Rank IC | Turnover |
|------|------|---------|----------|
| 动量 30d | 0.05 | 0.07 | 80% |
| MVRV | -0.04 | -0.06 | 30% |
| 波动率 | -0.03 | -0.04 | 50% |
| Funding | -0.04 | -0.05 | 70% |
| 活跃地址 | 0.03 | 0.04 | 40% |
| 情绪 | -0.03 | -0.04 | 60% |
| **综合** | **0.10** | **0.12** | **50%** |

**收益**：
- Long Top 5 - Short Bottom 5
- 年化 40-60%
- Sharpe 1.5-2.0
- 换手 50%/月

## 五、跨资产信号

### 1. 跨市场信号

**BTC vs 黄金**：
- BTC 与黄金相关性：~0.30
- 极端时点负相关（避险）
- "数字黄金"叙事

**BTC vs 美元指数**：
- 负相关 ~-0.45
- 美元升值 → BTC 承压
- 美联储政策传导

**BTC vs 美债**：
- 与 10Y 实际利率负相关
- 利率上升 → 资金从 BTC 流出
- **2022 年典型案例**

**ETH vs NASDAQ**：
- 正相关 ~0.65
- 科技股情绪传导
- 高利率压制

### 2. 跨链信号

**ETH/BTC 比率**：
- 反映资金轮动
- 比率上升 → 资金流入 ETH
- 比率下降 → 资金流入 BTC

**DeFi TVL**：
- 总锁仓量
- 上升 → DeFi 板块热
- 下降 → 风险

**Stablecoin 供应**：
- USDT + USDC 总市值
- 上升 → 弹药充足 → 风险资产利好
- 下降 → 资金撤离

### 3. 衍生品信号

**永续 Funding**：
- 极端 Funding → 反向
- Funding Rate 与现货价格强相关

**期权 Put/Call 比率**：
- P/C > 1：看空情绪
- P/C < 0.7：看多情绪
- 反向信号

**期权 IV Skew**：
- 25-delta Risk Reversal
- 看跌偏斜 → 恐慌
- 看涨偏斜 → 贪婪

### 4. 链上 vs 链下信号

**交易所 BTC 余额**：
- 余额下降 → 提币 → 长期持有
- 余额上升 → 准备卖出
- **2022 LUNA 崩盘前**：交易所余额异常上升

**稳定币 在交易所余额**：
- 上升 → 购买力（看多）
- 下降 → 资金外流（看空）

**巨鲸动向**：
- 1000+ BTC 移动
- 巨鲸地址分类（交易所、矿工、老钱）
- 实时监控（Whale Alert）

## 六、因子衰减与监控

### 1. 因子衰减

**典型模式**：
- 新因子：Sharpe 3+（可能过拟合）
- 半年后：Sharpe 1.5-2.0（真实）
- 1-2 年后：Sharpe 0.5-1.0（开始失效）
- 多年后：Sharpe 0.2-0.5（接近市场）

**加密因子特别快**：
- 2020-2021：meme 季，简单动量 Sharpe 5+
- 2022-2023：因子失效
- 2024-2025：复苏但竞争激烈

### 2. 因子监控指标

**实时监控**：
- IC（信息系数）
- Rank IC
- 因子收益的 Sharpe
- Turnover
- Long-Short 收益

**告警**：
- IC 绝对值 < 0.02 持续 3 个月
- 因子收益连续亏损
- 因子收益异常波动

### 3. 因子退役

**退役时机**：
- IC 持续 6 个月 < 0.01
- 因子收益年化 < 0
- 因子解释能力消失
- **替代因子出现**

**退役流程**：
1. 标记为"衰减"
2. 减小权重
3. 持续观察 3 个月
4. 完全移除

### 4. 因子发现

**新因子来源**：
- 学术文献
- 行业研究（Glassnode, Nansen）
- 实盘经验
- 数据科学（机器学习自动发现）

**因子评估**：
- 多时段回测
- 多币种测试
- 极端行情测试
- 逻辑支撑

## 八、加密因子研究前沿

### 1. DeFi 因子

**TVL Growth**：
- 过去 30 天 TVL 增长率
- 上升 → 协议基本面好
- 下降 → 资金外流

**协议收入**：
- 收入增长率
- P/E 类似（费用/收入）
- Quality 因子

**用户留存**：
- 月活用户 vs 新增用户
- 留存率
- 类似 SaaS 指标

### 2. NFT 因子

**稀有度**：
- 稀有度评分
- 类似规模因子

**持仓集中度**：
- 持有者数量
- 集中度越低越好
- 类似分散度因子

**交易频次**：
- 高交易频次 → 流动性好
- 类似流动性因子

### 3. L2 因子

**TVL vs 链上费用**：
- 类似传统 P/E
- 低费用高 TVL → 性价比高

**跨链桥流量**：
- 资金流入链
- 类似资金流因子

**活跃度**：
- 每日交易数
- 类似 NVT

### 4. AI/ML 因子
