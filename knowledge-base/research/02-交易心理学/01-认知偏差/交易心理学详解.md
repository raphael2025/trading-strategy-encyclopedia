## 概述

交易心理学是行为金融学在交易领域的应用，核心研究**人类认知偏差和情绪**如何系统性地导致交易者做出非理性决策。研究表明，**80% 以上的散户交易者长期亏损**，主要原因不是技术分析不足，而是心理因素。

Kahneman 在《Thinking, Fast and Slow》(2011) 中提出的**双系统理论**是理解交易心理的基石：
- **系统 1**：快速、直觉、情绪化（占日常决策 95%）
- **系统 2**：慢速、理性、需要努力（占交易决策 5%）

Mark Douglas 在《Trading in the Zone》(2000) 提出的**五项市场根本法则**：
1. 任何事都可能发生（市场无确定性）
2. 利润无限、风险有限（截断亏损）
3. 市场随机分布（无连续模式）
4. 每个时刻都是独特时刻
5. 交易者要"无我"地跟随市场

## 12 种核心交易认知偏差

### 1. 损失厌恶（Loss Aversion）

**定义**：亏损带来的痛苦强度约为同等盈利带来快乐的 **2-2.5 倍**（Tversky & Kahneman 1979 前景理论）

**案例**：
- BTC 在 60,000 跌到 58,000 让你痛苦的程度，超过 BTC 从 60,000 涨到 62,000 让你开心的程度
- 散户**过早止盈**、**过久扛损**几乎全是损失厌恶的产物

**对策**：
- 预设止损单（不让情绪介入）
- 把每笔交易看作**概率游戏**而非"必须赢"
- 接受"小亏"是"持续交易"的入场费

### 2. 处置效应（Disposition Effect）

**定义**：投资者倾向**过早卖出盈利的仓位、过久持有亏损的仓位**（Shefrin & Statman 1985）

**实证数据**：
- Odean (1998) 研究美国券商 10,000+ 账户：66% 的盈利仓位被过早止盈，而亏损仓位持有时间平均长 30%
- Frazzini (2006) 研究显示**反向应用**此偏差（快速止损、让利润奔跑）能产生 5-10% 年化超额收益

**对策**：
- 设定**非对称止盈/止损**（如盈利 2:1 风险回报比才出场）
- 写交易日志强制记录为什么止盈/止损

### 3. 锚定效应（Anchoring）

**定义**：决策过度依赖接收到的**第一个信息**（"锚"）

**交易中的表现**：
- "我买的时候是 1,000 美元"——把这个价格当作心理锚点
- 看分析师说"BTC 这次会到 100,000"，下意识就以 100,000 作为价格目标
- 永远忘不了"历史最高价"

**案例**：
- ETH 在 4,800 买的，现在 2,000，一直想"等回到 4,800 再卖"——但市场根本不在乎你的成本

**对策**：
- 关注**当前价格结构**（支撑/阻力、趋势线），而非历史买入价
- 重新审视每个交易决策："如果我现在没有这个仓位，我会在这个价格入场吗？"

### 4. 过度自信（Overconfidence）

**定义**：人们系统性**高估自己的知识、判断和控制力**（Svenson 1981）

**经典数据**：
- **80% 的司机**自评驾驶技术"高于平均"（Svenson 研究）
- 90% 的散户认为自己的选股能力超过市场平均

**对策**：
- 记录每笔交易的胜率、盈亏比
- 用**夏普比率、最大回撤**等客观指标评估
- 寻找"反向指标"——当所有人一致看多时警惕

### 5. 确认偏差（Confirmation Bias）

**定义**：人倾向**只寻找支持自己观点的信息**，忽略反对信息

**交易案例**：
- 看多 BTC 后，只看 bullish 的推文和 KOL，忽略任何看空分析
- 每次技术指标给出卖出信号时，找"这次不同"的理由

**对策**：
- **强制双面分析**：每笔交易写下"如果方向相反我会怎么操作"
- 关注**反对你仓位**的 KOL
- 设定"反向思考"检查清单

### 6. 沉没成本谬误（Sunk Cost Fallacy）

**定义**：因**已经投入**的时间/金钱/精力而坚持错误决策

**又叫"协和谬误"**：英国和法国政府因已投入 14 年坚持协和飞机项目

**交易表现**：
- 已经被套 50% 的山寨币，不断加仓"摊薄成本"
- 用了 3 个月研究的策略，亏损后不肯放弃

**对策**：
- 评估决策时**只看未来**，不看过去的投入
- 问自己："如果我现在空仓，会在这里买入吗？"
- 设定严格的"认错点"

### 7. 羊群效应 / 从众心理（Herding）

**定义**：倾向**跟随群体行为**，认为"这么多人不会错"

**著名案例**：
- 17 世纪荷兰郁金香泡沫
- 2000 年互联网泡沫
- 2021 年 GameStop 轧空
- 加密各类 ICO 狂热

**对策**：
- **逆向思考**：当所有人都说"这次不一样"时要警惕
- 关注**成交量和持仓**而非叙事
- 保留 20% 现金，在群体恐慌时加仓

### 8. 赌博者谬误（Gambler's Fallacy）

**定义**：错误地认为**过去事件**会影响**独立的未来事件**

**交易表现**：
- "BTC 连续 5 天跌了，明天该涨了"
- "我已经连亏 5 单了，下一单一定会赚"
- 实际上每次交易都是独立事件

**对策**：
- 理解**统计独立性**：5 次亏损不增加下次盈利概率
- 关注**期望值**而非"运气"

### 9. 可得性启发法（Availability Heuristic）

**定义**：根据**信息回忆的容易程度**判断事件概率

**交易表现**：
- 看到某币种 24 小时暴涨 50%，就认为这种机会很常见
- 媒体大量报道交易所被黑，就觉得"中心化交易所都不安全"
- 最近爆雷的项目让你高估"项目跑路"的概率

**对策**：
- 用**基础概率**（Base Rate）而非最近印象
- 关注**长期数据**而非短期事件

### 10. 代表性启发法（Representativeness Heuristic）

**定义**：根据**相似性**判断概率，忽视基础概率

**交易表现**：
- "这个项目像以太坊，会涨 1000 倍"——但基础概率是 99% 山寨币归零
- "这个 K 线形态像 2017 年顶部"——但市场背景完全不同

**对策**：
- 重视**基础概率**
- 区分"看起来像"和"本质相同"

### 11. 禀赋效应（Endowment Effect）

**定义**：拥有某物后**赋予它更高价值**（Thaler 1980）

**交易表现**：
- 持有一个币 1 年，估价 100 美元；同币种没持有估价 50 美元
- 不愿卖出盈利的仓位（"再拿拿"）
- 给亏损仓位"加仓摊薄"而不是直接止损

**对策**：
- 训练自己用"如果空仓"视角评估
- 写交易日志区分"基本面判断"和"情感"

### 12. 乐观偏差 / 乐观主义（Optimism Bias）

**定义**：系统性地**低估负面事件概率**

**交易表现**：
- "杠杆 10x 不会爆仓，我有止损"——结果极端行情打穿止损
- "DeFi 协议被攻击是小概率"——但 2022 年几乎月月都有
- "项目方不会跑路"——但 2021-2022 年 rug pull 频发

**对策**：
- 关注**历史爆仓、清算事件**频率
- 使用**压力测试**：假设最坏情况
- 始终使用**保守的杠杆**（如 ≤3x）

## 加密市场特有情绪现象

### FOMO（Fear of Missing Out，错失恐惧症）

**起源**：McGinnis 2004 商业研究
**加密含义**：看到别人暴富后，**在高点追入**导致接盘

**典型场景**：
- 看到某 meme 币涨 100 倍后追入
- 看到 BTC 突破历史新高，慌着追多
- "再不上车就晚了"

**数据**：Binance Research 2023 报告显示，超过 **60% 的散户首次买入** 发生在某币种 24 小时涨幅 > 30% 之后

**对策**：
- **冷却期规则**：大涨后等 48 小时再决定
- FOMO 时强制**降低仓位 50%**
- 把 FOMO 转化为 "已经错过 vs 还有无数机会" 的认知

### FUD（Fear, Uncertainty, Doubt）

**起源**：Amdahl 1975 商业策略
**加密含义**：被负面信息**恐慌性抛售**

**典型场景**：
- 监管消息一出立即清仓
- 大户砸盘谣言后跟风抛售
- "再不止损要归零了"

**数据**：Glassnode 数据显示，FUD 导致的抛售**80% 发生在反弹前 24 小时**

**对策**：
- **区分事实 vs 情绪**：先看原始来源
- 设定"新闻买入/卖出"规则
- 关注**链上数据**而非媒体头条

### 幸存者偏差（Survivorship Bias）

**加密表现**：
- 只看到 BTC、ETH 暴富，看不到 99% 归零币种
- 听到 KOL 晒 100x 收益，看不到他们过去爆仓的故事
- "我也能做到"——但样本中**只有 1% 活下来**

**对策**：
- 了解**失败案例**多于成功案例
- 关注**风险案例**而非暴富故事

### 杠杆强平连锁反应

**加密特有现象**：
- 高杠杆 + 集中持仓 → 价格下跌 → 强平 → 进一步下跌 → 连锁爆仓
- 2021 年 5 月 19 日 BTC 从 43,000 跌到 30,000 主要是连锁爆仓

**心理影响**：
- FOMO 加杠杆 → 被强平 → 损失厌恶 → 急于翻本 → 更大仓位 → 再爆
- 著名的"**爆仓螺旋**"

**对策**：
- 永远 ≤ 3x 杠杆
- 维持**低保证金率**（账户价值 / 持仓价值 ≥ 30%）
- 设定**减仓提醒**而非加仓

### 社群情绪放大效应

**加密特色**：
- Twitter、Telegram、Discord 群组实时放大情绪
- 1 个 100 万粉 KOL 喊单 → 散户 1 分钟内集体行动
- 社群**回音室效应**让偏差加剧

**对策**：
- **减少情绪来源**：调低推送通知
- 多元化信息源
- 设置"信息消化时间"：看到消息后**至少等 1 小时**再交易

### Rug Pull 恐慌

**DeFi/ICO 特有问题**：
- 项目方突然抽走流动性
- 2021-2022 年损失 **数十亿美元**

**心理影响**：
- 一旦听 rug pull → 恐慌卖出所有山寨币
- 误伤优质项目

**对策**：
- 关注**流动性锁定**和**项目方历史**
- 多样化持仓，不重仓单一项目
- 用**小额分散**降低单点风险

## 经典交易心理学框架

### Mark Douglas 五项市场法则详解

1. **任何事都可能发生**（Anything can happen）
   - 接受市场无确定性
   - 不为"意外"感到震惊

2. **利润无限、风险有限**（You don't need to know what's going to happen next to make money）
   - 重点是**风险管理**而非预测
   - 一致性执行 > 完美预测

3. **市场随机分布**（There is a random distribution between wins and losses）
   - 任何交易**独立**
   - 连亏 5 次不增加下次盈利概率

4. **每个时刻都是独特**（Every moment in the market is unique）
   - 没有"完全相同"的两次交易
   - 机械套用历史模式会失败

5. **无我跟随市场**（Trade what you see, not what you think）
   - 区分**主观愿望**和**客观信号**
   - "我想要 BTC 涨" vs "K 线显示下跌趋势"

### Brett Steenbarger 交易员心理学四要素

《Trading Psychology 2.0》提出：
1. **自我觉察**（Self-awareness）：知道自己的状态
2. **自律**（Self-discipline）：执行交易计划
3. **情绪调节**（Emotional regulation）：不被情绪驱动
4. **持续学习**（Continuous learning）：从每笔交易中成长

### Van Tharp 交易者心理模型

《Trade Your Way to Financial Freedom》提出：
- **交易系统**（入场/出场/仓位）
- **交易心理**（执行系统）
- **资金管理**（仓位规模）
三者缺一不可。

**关键洞察**：即使是最好的系统，心理素质差的人也会破坏它。

## 行为金融学实证研究

### Odean (1998) 处置效应研究
- 样本：1991-1996 年美国 10,000 个券商账户
- 核心发现：**66% 的盈利仓位被卖出** vs **亏损仓位持有时间长 30%**
- 结论：处置效应是系统性的、非理性的

### Barber & Odean (2000) 性别与交易
- 男性交易频率**高 45%** 而收益**低 6.5%/年**
- 过度自信导致频繁交易，收益更低

### Kahneman & Riepe (1998) 投资者心理学
- 投资决策中**情绪权重**远大于分析权重
- 投资者应**设计规则**让系统做决定

### Frazzini (2006) 反向处置效应
- 应用"快速止损、让利润奔跑"
- 长期超额收益 **5-10% 年化**
- 验证了处置效应是**可利用的偏差**

### 加密市场专项研究
- **Binance Research (2023)**: 60%+ 散户首次买入在币种 24h 涨 30% 后
- **Glassnode (2022)**: FUD 抛售 80% 发生在反弹前 24 小时
- **Chainalysis (2023)**: rug pull 2022 年损失超 **30 亿美元**
- **Coinglass**: 2021-2022 杠杆强平损失超 **1000 亿美元**

## 实战应用

### 交易前心理检查清单

每笔交易前问自己：
- [ ] 我是基于**信号**还是**感觉**？
- [ ] 如果我没这个仓位，我会在这个价格入场吗？
- [ ] 我能接受**最坏情况**吗？
- [ ] 这笔交易**占用总资金比例**是多少？
- [ ] 我的**止损点**在哪里？

### 持仓期间纪律

- [ ] 不看消息面**只看价格行为**
- [ ] 移动止损**保护利润**
- [ ] 每日**回顾**一次（不超过 1 次）
- [ ] 记录**情绪状态**

### 亏损后的应对

1. **暂停交易 24 小时**（避免赌徒心态）
2. **复盘**：是系统问题还是执行问题？
3. **降低仓位** 50% 一周
4. **写交易日志**：为什么亏损？下次怎么避免？
5. **不接受"运气不好"** 的解释——总有原因

### 盈利后的应对

1. **提取部分利润**（保护本金）
2. **记录成功原因**（可复制的模式）
3. **不要立即加仓**（避免过度自信）
4. **保持原仓位规模**，避免"现在我行了"

## 工具与练习

### 推荐练习

#### 1. 交易日志（必做）
记录每笔交易：
- 入场理由（具体信号）
- 持仓时情绪（焦虑？兴奋？平静？）
- 出场理由
- 实际结果 vs 预期
- 复盘：什么做对/做错？

#### 2. 情绪打分（1-10）
- 入场前、持仓中、出场后各打分
- 识别**情绪触发点**

#### 3. 模拟交易 3 个月
- 在**不冒真金**的情况下练习
- 验证系统的心理可行性

#### 4. 冥想 / 正念训练
- 每日 10 分钟正念冥想
- 研究显示能**降低焦虑 30%+**
- 提高**情绪调节能力**

#### 5. 阅读经典著作
- 《Trading in the Zone》- Mark Douglas
- 《Trading Psychology 2.0》- Brett Steenbarger
- 《Trade Your Way to Financial Freedom》- Van Tharp
- 《Thinking, Fast and Slow》- Kahneman
- 《Misbehaving》- Richard Thaler

## A.1 处置效应 (Disposition Effect) 测度

```python
"""
Disposition Effect (Shefrin & Statman 1985)
= 倾向卖出盈利单 + 持有亏损单
- 测度：Odean (1998) 方法
  PGR (Proportion of Gains Realized) = 卖出盈利单 / 总盈利单
  PLR (Proportion of Losses Realized) = 卖出亏损单 / 总亏损单
  DE = PGR - PLR  范围 [-1, 1]，正值越强越处置效应
"""
import pandas as pd
import numpy as np


def disposition_effect(trades: pd.DataFrame) -> dict:
    """
    trades: DataFrame 含 ['trade_id', 'action' (buy/sell),
                          'px', 'qty', 'cost_basis', 'ts']
    返回 {PGR, PLR, DE}
    """
    sells = trades[trades['action'] == 'sell'].copy()
    sells['pnl'] = (sells['px'] - sells['cost_basis']) * sells['qty']
    sells['is_gain'] = sells['pnl'] > 0

    pgr = sells['is_gain'].mean() if len(sells) else 0
    losing = sells[~sells['is_gain']]
    plr = 1 - losing['is_gain'].mean() if len(losing) else 0
    de = pgr - plr
    return {"PGR": pgr, "PLR": plr, "DE_score": de,
            "interpretation": "strong" if de > 0.3 else
                              "moderate" if de > 0.1 else "weak"}
```

## A.2 损失厌恶实验（Prospect Theory 拟合）

```python
"""
Kahneman-Tversky 1979 Prospect Theory 拟合
- 价值函数: v(x) = x^α  if x >= 0, -λ(-x)^α  if x < 0
- 概率加权函数: w(p) = p^γ / (p^γ + (1-p)^γ)^(1/γ)
- 典型参数: α ≈ 0.88, λ ≈ 2.25, γ ≈ 0.61
"""
import numpy as np
from scipy.optimize import minimize


def prospect_value(outcomes: np.ndarray, probs: np.ndarray,
                   alpha: float = 0.88, lam: float = 2.25,
                   gamma: float = 0.61) -> float:
    """计算期望前景值"""
    v = np.where(outcomes >= 0, outcomes**alpha, -lam * (-outcomes)**alpha)
    w = probs**gamma / (probs**gamma + (1 - probs)**gamma) ** (1/gamma)
    return np.sum(v * w)


def fit_prospect_theory(choices: pd.DataFrame) -> dict:
    """
    choices: 实验数据
    列：['g_outcome', 'g_prob', 'g_certain', 'l_outcome', 'l_prob', 'chosen']
    """
    def loss(params):
        alpha, lam, gamma = params
        if not (0 < alpha < 1) or lam <= 0 or not (0 < gamma < 1):
            return 1e10
        err = 0
        for _, row in choices.iterrows():
            v_certain = row['g_certain']
            v_gamble = prospect_value(
                np.array([row['g_outcome'], row['l_outcome']]),
                np.array([row['g_prob'], row['l_prob']]),
                alpha, lam, gamma)
            err += (v_certain - v_gamble) ** 2
        return err

    res = minimize(loss, [0.88, 2.25, 0.61], method='Nelder-Mead')
    return dict(zip(['alpha', 'lambda', 'gamma'], res.x))
```

## A.3 锚定效应检测

```python
"""
锚定效应 = 决策时过度依赖最初给定的参考值
- 加密场景：常锚定历史高点/低点
- 检测：买卖价相对历史极值的偏离 + 实际波动率
"""
import pandas as pd
import numpy as np


def anchoring_intensity(price: pd.Series,
                        reference: float,
                        realized_vol: float) -> float:
    """
    锚定强度 = |current - reference| / (realized_vol * N)
    值小 = 强锚定
    """
    return abs(price.iloc[-1] - reference) / (realized_vol * 20)


# 心理账户分析
def mental_accounting_loss(portfolio_pnl: float,
                           opportunity_pnl: float) -> float:
    """
    mental_acct_score = 实际 PnL 与"机会成本"差距
    - 实际 + 机会也 + → 心情好
    - 实际 - 但 机会 + → 极度遗憾（mental accounting 强）
    """
    if opportunity_pnl < 0 <= portfolio_pnl:
        return 0.0
    if portfolio_pnl < 0 < opportunity_pnl:
        return 1.0  # 强 mental accounting
    return 0.0
```

## A.4 过度交易识别

```python
"""
Overtrading = 交易频率超过 alpha 含量
- 测度：实际换手率 vs 风格基准换手率
"""
import pandas as pd
import numpy as np


def overtrading_score(trades: pd.DataFrame,
                      portfolio_value: float,
                      benchmark_turnover: float = 1.0) -> float:
    """
    turnover = sum(abs(qty * px)) / portfolio_value
    """
    turnover = (trades['qty'] * trades['px']).abs().sum() / portfolio_value
    return turnover / benchmark_turnover
```

## A.5 风险偏好动态建模

```python
"""
动态风险偏好：
- 收益后 → 风险偏好上升 (House Money Effect)
- 亏损后 → 风险偏好下降 (Break-Even Effect)
"""
import numpy as np
import pandas as pd


def dynamic_risk_aversion(pnl_history: pd.Series,
                           base_lambda: float = 2.0) -> pd.Series:
    """
    近期 PnL 影响当前风险厌恶
    - 收益 → λ ↓（更冒险）
    - 亏损 → λ ↑（更保守）
    """
    cumret = pnl_history.cumsum()
    return base_lambda - 0.5 * np.tanh(cumret / pnl_history.std())
```

## A.6 认知偏差 cheat sheet

| 偏差 | 检测方法 | 应对 |
|---|---|---|
