## 一、期权基本概念

**期权（Option）**是一种金融衍生品，赋予买方在**特定时间以特定价格买卖基础资产**的**权利**，但不是义务。卖方收取权利金，承担相应义务。

### 1. 关键术语

| 术语 | 含义 | 示例 |
|------|------|------|
| **标的资产（Underlying）** | 期权所基于的资产 | BTC、ETH、SOL |
| **行权价（Strike）** | 未来可以买卖的价格 | BTC 70,000 Call |
| **到期日（Expiry）** | 期权有效的最后日期 | 2025-06-27 |
| **权利金（Premium）** | 买方支付给卖方的费用 | 0.05 BTC |
| **行权类型** | 美式 vs 欧式 | 欧式（到期日行权）|
| **合约规模** | 一份合约的标的数量 | BTC 0.1, ETH 1 |

### 2. 两大类型

**看涨期权（Call Option）**：
- 买方有权**以行权价买入**标的
- 看涨标的用，**最大亏损 = 权利金**
- 例：BTC 现 65,000，买入 70,000 Call，到期 BTC 涨到 80,000 → 盈利 (80,000-70,000) × 0.1 = 1,000 USD

**看跌期权（Put Option）**：
- 买方有权**以行权价卖出**标的
- 看跌标的用，**最大亏损 = 权利金**
- 例：BTC 现 65,000，买入 60,000 Put，到期 BTC 跌到 50,000 → 盈利 (60,000-50,000) × 0.1 = 1,000 USD

### 3. 期权的价值构成

**理论价值 = 内在价值 + 时间价值**

- **内在价值（Intrinsic Value）**：
  - Call: max(标的价格 - 行权价, 0)
  - Put: max(行权价 - 标的价格, 0)
  - 例：BTC 70,000，行权价 65,000 Call → 内在价值 5,000

- **时间价值（Time Value）**：
  - 期权费 - 内在价值
  - 距离到期日越远，时间价值越大
  - 临近期权到期日时，时间价值加速衰减（Theta 衰减）

## 二、Black-Scholes 模型

### 1. 模型原理

**Black-Scholes-Merton (1973)** 是期权定价的奠基性模型，1997 年获诺贝尔经济学奖。

**5 个假设**：
1. 市场无摩擦（无税、无交易成本）
2. 标的资产价格服从对数正态分布
3. 无风险利率恒定
4. 欧式期权（到期日才能行权）
5. 标的资产可任意分割、可卖空

**核心公式**（Call）：
```
C = S × N(d1) - K × e^(-rT) × N(d2)

d1 = [ln(S/K) + (r + σ²/2)T] / (σ√T)
d2 = d1 - σ√T

其中：
S = 标的价格
K = 行权价
r = 无风险利率
σ = 隐含波动率
T = 到期时间（年化）
N() = 标准正态分布累积函数
```

### 2. 关键参数解读

| 参数 | 影响方向 | 解释 |
|------|---------|------|
| 标的价格 S↑ | Call↑, Put↓ | 价格上行利好看涨 |
| 行权价 K↑ | Call↓, Put↑ | 行权价高不利于看涨 |
| 隐含波动率 σ↑ | Call↑, Put↑ | 波动大，期权更贵 |
| 利率 r↑ | Call↑, Put↓ | 折现率影响 |
| 到期时间 T↑ | 期权费↑ | 时间价值 |

### 3. 模型局限

- **波动率微笑**（Volatility Smile）：BSM 假设波动率恒定，实际中行权价偏离越远，隐含波动率越高
- **肥尾分布**：加密资产极端事件多，BSM 假设的"对数正态分布"低估了极端事件概率
- **跳跃过程**：加密市场常出现瞬间跳空（CME gap、链上事件），BSM 无法捕捉
- **可卖空假设**：加密市场部分代币难以卖空

**改进模型**：
- **Heston 模型**（1993）：随机波动率
- **Merton 跳跃-扩散模型**（1976）：加入跳跃项
- **SABR 模型**：用于利率衍生品
- **Local Volatility**：Dupire 公式

### 4. 隐含波动率（IV）与历史波动率（HV）

**IV（隐含波动率）**：
- 由期权市场价格反推
- 代表市场对未来波动的预期
- 常被称为"恐慌指标"

**HV（历史波动率）**：
- 过去 N 天实际波动率
- 通常用对数收益率标准差年化

**关系**：
- **IV > HV**：期权偏贵
- **IV < HV**：期权便宜
- **IV-HV 差**：波动率风险溢价

**加密期权典型 IV 范围**（2024）：
- BTC 期权：30-80%（平均 50%）
- ETH 期权：40-100%（平均 65%）
- 山寨币期权：60-200%

## 四、期权盈亏分析（P&L）

### 1. 单一期权

**买入 Call（Long Call）**：
- 最大亏损：权利金
- 最大盈利：∞
- 盈亏平衡点：行权价 + 权利金

**卖出 Call（Short Call）**：
- 最大盈利：权利金
- 最大亏损：∞
- 风险：标的价格暴涨

**买入 Put（Long Put）**：
- 最大亏损：权利金
- 最大盈利：行权价 × 1
- 盈亏平衡点：行权价 - 权利金

**卖出 Put（Short Put）**：
- 最大盈利：权利金
- 最大亏损：行权价 × 1
- 风险：标的价格暴跌

### 2. 实战案例

**案例 1：BTC 现 65,000，预期下跌**

策略：买入 60,000 Put，权利金 1,000 USD，到期日 30 天

| BTC 30 天后价格 | 期权价值 | 盈亏 |
|----------------|---------|------|
| 70,000 | 0 | -1,000 |
| 65,000 | 0 | -1,000 |
| 60,000 | 0 | -1,000 |
| 55,000 | 5,000 | +4,000 |
| 50,000 | 10,000 | +9,000 |

**盈亏平衡**：59,000（行权价 - 权利金）

**案例 2：ETH 现 3,500，认为会横盘**

策略：卖出 3,800 Call（收 50 USD）+ 卖出 3,200 Put（收 50 USD）
- **Short Strangle**：收 100 USD 权利金
- 风险：ETH 突破 3,900 或跌破 3,100
- 收益：ETH 维持在 3,200-3,800 区间 → 净赚 100 USD

## 五、期权在加密市场的应用

### 1. 风险管理

**套保（Hedging）**：
- 现货多头 + 买入 Put = 保险
- 矿工对冲：用 Put 锁定卖出价
- 长期持有者用 Put 防御暴跌

**收益增强**：
- 现货 + 卖出 Call（Covered Call）
- 在 Deribit 等平台出售 BTC 现货并卖出 OTM Call
- 长期熊市可降低持仓成本

### 2. 投机

- **方向性投机**：用期权获得杠杆（成本低）
- **波动率投机**：跨式/宽跨式（Straddle/Strangle）
- **方向+波动率**：蝶式、铁鹰

### 3. 套利

- **期现套利**：期权 IV 与 HV 的差异
- **跨交易所套利**：Deribit vs Binance
- **波动率曲面套利**：不同到期日的 IV 差异

### 4. 机构用例

- **Jump Crypto、Cumberland、Wintermute**：做市商
- **Galaxy Digital、Polychain、Paradigm**：基金对冲
- **CME 微型 BTC/ETH 期货期权**：传统机构入口
- **MicroStrategy 等公司**：用期权对冲 BTC 持仓

## 六、关键概念：Moneyness

| 状态 | Call | Put |
|------|------|-----|
| **ITM (In-the-Money)** | 标的价格 > 行权价 | 标的价格 < 行权价 |
| **ATM (At-the-Money)** | 标的价格 ≈ 行权价 | 标的价格 ≈ 行权价 |
| **OTM (Out-of-the-Money)** | 标的价格 < 行权价 | 标的价格 > 行权价 |

**流动性最佳**：ATM 期权（行权价接近标的价格）

## A.1 Black-Scholes 欧式期权定价

```python
"""
Black-Scholes 模型（无股息版）— 适用股票/不付息币种
公式：Call = S*N(d1) - K*exp(-rT)*N(d2)
      Put  = K*exp(-rT)*N(-d2) - S*N(-d1)
      d1 = (ln(S/K) + (r + σ²/2)T) / (σ√T)
      d2 = d1 - σ√T
"""
import numpy as np
from scipy.stats import norm


def bs_price(S: float, K: float, T: float, r: float, sigma: float,
             option_type: str = "call") -> float:
    """
    S: 标的价格；K: 行权价；T: 到期时间（年）；r: 无风险利率；sigma: 波动率
    option_type: 'call' 或 'put'
    """
    if T <= 0:
        return max(0.0, S - K) if option_type == "call" else max(0.0, K - S)
    d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    if option_type == "call":
        return S * norm.cdf(d1) - K * np.exp(-r * T) * norm.cdf(d2)
    return K * np.exp(-r * T) * norm.cdf(-d2) - S * norm.cdf(-d1)


# 单元测试
if __name__ == "__main__":
    # BTC 价 60,000，行权 65,000，30 天，r=5%，σ=60%
    c = bs_price(60000, 65000, 30/365, 0.05, 0.60, "call")
    p = bs_price(60000, 65000, 30/365, 0.05, 0.60, "put")
    print(f"Call={c:.2f}  Put={p:.2f}")
    # Call≈1900, Put≈6300（OTM call / ITM put）
```

## A.2 Black-76 期货期权定价（适用 BTC/ETH 永续）

```python
"""
Black-76 模型（期货/远期期权）
- 加密永续没有真期货，但用 forward price 代替
- 公式：Call = exp(-rT) * [F*N(d1) - K*N(d2)]
        d1 = ln(F/K) / (σ√T) + σ√T/2
"""
import numpy as np
from scipy.stats import norm


def black76_price(F: float, K: float, T: float, r: float, sigma: float,
                  option_type: str = "call") -> float:
    """F: 远期价格（永续 → 现货 * exp((funding_apr)*T)）；其他同 BS"""
    if T <= 0:
        intrinsic = max(0.0, F - K) if option_type == "call" else max(0.0, K - F)
        return np.exp(-r * T) * intrinsic
    d1 = (np.log(F / K) + 0.5 * sigma**2 * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    disc = np.exp(-r * T)
    if option_type == "call":
        return disc * (F * norm.cdf(d1) - K * norm.cdf(d2))
    return disc * (K * norm.cdf(-d2) - F * norm.cdf(-d1))
```

## A.3 Put-Call Parity 套利监测

```python
"""
Put-Call Parity: C - P = S - K*exp(-rT)
- 加密市场 Deribit 上偶有偏离 → 套利机会
- 偏离阈值：0.1% of underlying 即可触发（扣手续费后）
"""
import numpy as np


def parity_deviation(C: float, P: float, S: float, K: float,
                     T: float, r: float) -> tuple[float, str]:
    """
    返回 (偏离百分比, 信号)
    信号 ∈ {'call_expensive', 'put_expensive', 'fair'}
    """
    theoretical = C - P - (S - K * np.exp(-r * T))
    pct = theoretical / S * 100
    if pct > 0.05:
        signal = "call_expensive"  # 卖 Call 买 Put + 卖现货
    elif pct < -0.05:
        signal = "put_expensive"   # 买 Call 卖 Put + 买现货
    else:
        signal = "fair"
    return pct, signal
```

## A.4 隐含波动率 Newton-Raphson 求解

```python
"""
从市场期权价格反推 σ
- Newton-Raphson: σ_{n+1} = σ_n - (BS(σ_n) - C_mkt) / vega(σ_n)
- vega = S * √T * N'(d1)
- 收敛阈值：|C_mkt - BS| < 1e-6
"""
import numpy as np
from scipy.stats import norm


def implied_vol(C_mkt: float, S: float, K: float, T: float, r: float,
                option_type: str = "call", tol: float = 1e-6,
                max_iter: int = 100) -> float:
    """返回隐含波动率（年化）；若不收敛返回 NaN"""
    if T <= 0 or C_mkt <= 0:
        return np.nan
    sigma = 0.5  # 初始猜测
    for _ in range(max_iter):
        c = bs_price(S, K, T, r, sigma, option_type)
        diff = c - C_mkt
        if abs(diff) < tol:
            return sigma
        d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
        vega = S * np.sqrt(T) * norm.pdf(d1)
        if vega < 1e-9:
            return np.nan
        sigma = sigma - diff / vega
        sigma = max(0.001, min(sigma, 5.0))  # 限幅
    return np.nan  # 不收敛
```

## A.5 希腊字母 Greeks 解析公式

```python
"""
Greeks（不含股息版）
Delta Call = N(d1)           ; Delta Put = N(d1) - 1
Gamma = N'(d1) / (S σ √T)    ; Vega = S √T N'(d1)
Theta Call = -S σ N'(d1)/(2√T) - r K e^{-rT} N(d2)
Rho Call = K T e^{-rT} N(d2)
"""
import numpy as np
from scipy.stats import norm


def bs_greeks(S: float, K: float, T: float, r: float, sigma: float,
              option_type: str = "call") -> dict:
    if T <= 0:
        return {"delta": np.nan, "gamma": np.nan, "vega": np.nan,
                "theta": np.nan, "rho": np.nan}
    d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    pdf_d1 = norm.pdf(d1)
    disc = np.exp(-r * T)

    delta = norm.cdf(d1) if option_type == "call" else norm.cdf(d1) - 1
    gamma = pdf_d1 / (S * sigma * np.sqrt(T))
    vega = S * np.sqrt(T) * pdf_d1 / 100   # 1% 变动
    if option_type == "call":
        theta = (-S * sigma * pdf_d1 / (2 * np.sqrt(T))
                 - r * K * disc * norm.cdf(d2)) / 365   # 每天
        rho = K * T * disc * norm.cdf(d2) / 100         # 1% 变动
    else:
        theta = (-S * sigma * pdf_d1 / (2 * np.sqrt(T))
                 + r * K * disc * norm.cdf(-d2)) / 365
        rho = -K * T * disc * norm.cdf(-d2) / 100
    return {"delta": delta, "gamma": gamma, "vega": vega,
            "theta": theta, "rho": rho}

```

## A.6 Deribit 期权链实时拉取与 IV Surface 构建

```python
"""
Deribit 公开 API v2 拉取 BTC/ETH 期权链
- 不需要 API key（公开 endpoint）
- 文档：https://docs.deribit.com/
"""
import requests
import pandas as pd
import numpy as np


def fetch_deribit_option_chain(currency: str = "BTC", kind: str = "option") -> pd.DataFrame:
    """返回全部未到期期权（按到期日展开成行权价 × 类型的矩阵）"""
    url = f"https://www.deribit.com/api/v2/public/get_instruments"
    r = requests.get(url, params={"currency": currency, "kind": kind, "expired": False},
                     timeout=20)
    r.raise_for_status()
    rows = []
    for ins in r.json()["result"]:
        name = ins["name"]            # 如 BTC-27JUN25-65000-C
        parts = name.split("-")
        rows.append({
            "instrument": name,
            "currency": parts[0],
            "expiry": pd.to_datetime(parts[1], format="%d%b%y"),
            "strike": float(parts[2]),
            "type": "C" if parts[3] == "C" else "P",
        })
    return pd.DataFrame(rows)


def build_iv_surface(chain: pd.DataFrame, spot: float, risk_free: float = 0.05) -> pd.DataFrame:
    """
    对每条期权拉 mark_iv → 构建 (expiry, strike) 网格
    实际需配合 fetch_ticker 接口；此处展示结构
    """
    from A1 import bs_price  # 同文件内
    from A4 import implied_vol

    ivs = []
    for _, row in chain.iterrows():
        ticker = requests.get(
            "https://www.deribit.com/api/v2/public/ticker",
            params={"instrument_name": row["instrument"]},
            timeout=10,
        ).json()["result"]
        mark_price = ticker["mark_price"]
        iv = implied_vol(mark_price, spot, row["strike"],
                          (row["expiry"] - pd.Timestamp.utcnow().tz_localize(None)).days / 365,
                          risk_free, "call" if row["type"] == "C" else "put")
        ivs.append({**row.to_dict(), "iv": iv, "mark": mark_price})
    return pd.DataFrame(ivs)
```
