## 一、央行政策影响加密市场的 4 大传导渠道

### 1. 流动性渠道（最直接）

**机制**：央行加息/降息 → 美元/欧元/日元流动性 → 风险资产价格

**传导路径**：
- 央行加息 → 美元指数走强 → 风险资产承压
- 央行降息/扩表 → 流动性宽松 → 风险资产上升
- 量化紧缩（QT）→ 美元流动性收紧 → 加密市场承压
- 量化宽松（QE）→ 美元贬值预期 → 投资者寻求替代品（含 BTC）

### 2. 收益率渠道

**机制**：利率 → 实际收益率 → 持有无息资产（BTC）vs 债券的机会成本

- 实际利率（10Y TIPS）↑ → 持有 BTC 成本↑ → BTC 价格承压
- 实际利率↓ → 现金和债券吸引力↓ → BTC 黄金等替代资产↑
- **历史相关性**：BTC 价格与 10Y TIPS 实际利率的负相关，2018 年后趋强

### 3. 汇率渠道

**机制**：央行政策 → 本币汇率 → USDT 等稳定币的"美元化"流向

- 美元升值 → 新兴市场货币贬值 → USDT 需求↑（平行美元化）
- 美元贬值 → 资本流向风险资产 → 加密市场↑
- **2024 年**：日元/韩元/巴西雷亚尔贬值期间，USDT 在这些国家的 P2P 溢价高达 5-10%

### 4. 预期渠道

**机制**：央行前瞻指引 → 市场预期 → 提前定价

- 鲍威尔讲话 → BTC 当日波动 3-10%
- 美联储点阵图（Dot Plot）→ 影响 BTC 中期趋势
- "Higher for Longer"政策 → 加密市场承压
- 降息预期 → 加密市场反弹

## 二、关键政策事件与加密市场反应

### 1. 2020 年 COVID 危机

- **2020 年 3 月 15 日**：美联储紧急降息至 0-0.25%
- 启动**无限 QE**：每月购买 800 亿美元国债 + 400 亿美元 MBS
- **市场反应**：BTC 从 5,000 美元涨到 2021 年 11 月 69,000 美元（+1380%）
- **机制**：史无前例的流动性注入 → 通胀预期↑ → 投资者寻求"数字黄金"

### 2. 2021-2022 年通胀冲击

- 2021 年美国 CPI 最高 **9.1%**（2022 年 6 月）
- 鲍威尔 2022 年 3 月开始**激进加息**（首加 25bp）
- 2022 年 6 月：单次**加 75bp**（28 年来最大）
- 2022 年 11 月：联邦基金利率 4.00-4.25%（年内加 425bp）
- **2023 年 7 月**：终端利率 5.25-5.50%（22 年来最高）
- **市场反应**：BTC 从 69,000 跌至 15,500 美元（-78%）

### 3. 2023 年 "Higher for Longer"

- 美联储 2023 年 H2 维持 5.25-5.50%
- BTC 在 25,000-35,000 美元区间震荡
- **机制**：高利率环境下，杠杆交易成本高，机构兴趣受抑

### 4. 2024 年降息预期

- 2024 年 9 月 18 日：美联储**首次降息 50bp**（5.00-5.50% → 4.75-5.00%）
- 11 月：再降 25bp
- 12 月：再降 25bp
- **市场反应**：BTC 从 9 月 55,000 美元涨至 12 月 100,000 美元（+82%）
- **历史意义**：BTC 首次突破 10 万美元大关

### 5. 2025 年新格局

- 2025 年 1 月：暂停降息（评估特朗普关税政策）
- 特朗普"政府效率部"（DOGE 关联）推动财政紧缩
- 美联储面临**财政赤字货币化**压力

- 同期 BTC 涨幅 +1380%
- 黄金涨幅 +45%

- 限量供应（2100 万枚）

- 美元在全球外汇储备占比 58%（2024），较 2000 年 71% 下降
- BIS 警告：稳定币可能**进一步强化美元**

### 3. "数字黄金"vs"高贝塔科技股"

**现代研究**（2024）：
- BTC 与纳指相关性：0.65（流动性驱动）

- **结论**：BTC 是**混合资产**：40% 黄金 + 40% 科技股 + 20% 货币

## A.1 FRED 宏观数据拉取

```python
"""
FRED (Federal Reserve Economic Data) API
- 免费，需注册 API key (https://fred.stlouisfed.org/docs/api/api_key.html)
- 关键序列：M2, DXY, FEDFUNDS, CPIAUCSL, 10Y-2Y
"""
import requests
import pandas as pd
from typing import Optional


class FREDClient:
    BASE = "https://api.stlouisfed.org/fred/series/observations"

    def __init__(self, api_key: str):
        self.key = api_key

    def series(self, series_id: str, start: str = "2015-01-01") -> pd.Series:
        params = {
            'series_id': series_id,
            'api_key': self.key,
            'file_type': 'json',
            'observation_start': start,
        }
        r = requests.get(self.BASE, params=params, timeout=10)
        r.raise_for_status()
        obs = r.json()['observations']
        dates = [o['date'] for o in obs]
        values = [float(o['value']) if o['value'] != '.' else None for o in obs]
        s = pd.Series(values, index=pd.to_datetime(dates), name=series_id)
        return s.dropna()
```

## A.2 Nelson-Siegel-Svensson 利率曲线

```python
"""
Nelson-Siegel-Svensson (NSS) 拟合收益率曲线
- β0: 长期利率水平
- β1: 斜率 (- 表示向上倾斜)
- β2: 曲度
- β3: 第二曲度（Svensson 扩展）
- λ1, λ2: 衰减因子
"""
import numpy as np
from scipy.optimize import minimize
from scipy.optimize import differential_evolution


def nss(t, beta0, beta1, beta2, beta3, lambda1, lambda2):
    """NSS 即期利率模型"""
    t = np.asarray(t, dtype=float)
    factor1 = (1 - np.exp(-lambda1 * t)) / (lambda1 * t)
    factor2 = factor1 - np.exp(-lambda1 * t)
    factor3 = (1 - np.exp(-lambda2 * t)) / (lambda2 * t) \
              - np.exp(-lambda2 * t)
    return beta0 + beta1 * factor1 + beta2 * factor2 + beta3 * factor3


def fit_nss(maturities: np.ndarray, yields: np.ndarray) -> dict:
    """最小化 NSS 拟合误差"""
    def loss(params):
        b0, b1, b2, b3, l1, l2 = params
        if l1 <= 0 or l2 <= 0:
            return 1e10
        return np.sum((nss(maturities, *params) - yields) ** 2)

    bounds = [(0, 15), (-15, 15), (-15, 15), (-15, 15), (0.01, 5), (0.01, 5)]
    result = differential_evolution(loss, bounds, maxiter=200, seed=42)
    return dict(zip(['beta0', 'beta1', 'beta2', 'beta3', 'lambda1', 'lambda2'],
                    result.x))
```

## A.4 BTC 与 M2 滚动相关

```python
"""
BTC 价格 vs 全球 M2 同比
- 通常相关系数 0.3-0.5，时滞 6-9 个月
- 滚动 12 月 / 24 月
"""
import pandas as pd
import numpy as np


def btc_m2_correlation(btc_monthly: pd.Series,
                       m2_monthly: pd.Series,
                       window: int = 12,
                       max_lag: int = 12) -> pd.DataFrame:
    """
    btc_monthly: 月频 BTC 价格
    m2_monthly: 月频 M2 供应
    window: 滚动月数
    max_lag: 最大滞后月数
    """
    btc_ret = btc_monthly.pct_change()
    m2_yoy = m2_monthly.pct_change(12)

    results = []
    for lag in range(0, max_lag + 1):
        m2_lagged = m2_yoy.shift(lag)
        corr = btc_ret.rolling(window).corr(m2_lagged)
        results.append(corr.rename(f"lag_{lag}m"))
    return pd.concat(results, axis=1)
```

## A.5 美元 DXY 影响力测度

```python
"""
DXY (美元指数) 与 BTC 反向关系
- 美元强 = 风险资产弱，BTC 跌
- 反之 BTC 涨

import numpy as np


def dxy_btc_beta(dxy_returns: pd.Series, btc_returns: pd.Series,
                 window: int = 90) -> pd.Series:
    """
    滚动 beta
    beta = cov(DXY, BTC) / var(DXY)
    """
    def _beta(x, y):
        if x.std() == 0:
            return 0
        return x.cov(y) / x.var()
    df = pd.DataFrame({'dxy': dxy_returns, 'btc': btc_returns})
    return df['btc'].rolling(window).corr(df['dxy'])


def regime_split(dxy_level: float, threshold: float = 100) -> str:
    """DXY > 100 = 强势美元"""
    return "DXY_up" if dxy_level > threshold else "DXY_down"
```

```python
def days_to_next_meeting(meeting_dates: list[datetime]) -> int:
    """距下次议息的天数"""
    now = datetime.utcnow()
    future = [d for d in meeting_dates if d > now]
    if not future:
        return -1
    return (min(future) - now).days
```
