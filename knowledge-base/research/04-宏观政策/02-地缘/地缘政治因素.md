## 一、地缘政治如何影响加密市场

加密货币自诞生起就具有**反监管、跨境、去中心化**的基因，使其天然成为**地缘政治博弈的工具**。三大影响路径：

1. **资本流动规避**：跨国资金绕过传统银行体系（SWIFT/CHIPS）
2. **制裁规避与执行**：稳定币 + DEX 成为制裁/反制裁的金融工具
3. **国家战略储备**：BTC 从"反政府资产"演变为"国家储备资产"

## 二、关键地缘事件与加密市场反应

### 1. 2017 年 ICO 浪潮 + 韩国"泡菜溢价"

- 韩国市场出现 30-50% 的"泡菜溢价"（Kimchi Premium），反映资金管制下民众抢购 BTC
- 2017 年 9 月中国全面禁止 ICO → 项目大量转向新加坡、瑞士
- 2018 年 1 月韩国考虑禁止加密交易 → 韩国交易员涌入日本交易所

### 2. 2021 年 9 月中国全面禁止

- **2021 年 9 月 24 日**：中国人民银行等 10 部门联合发布《关于进一步防范和处置虚拟货币交易炒作风险的通知》
- 全面禁止：虚拟货币相关业务活动、境外交易所向中国居民提供服务
- **市场反应**：BTC 当日下跌 5%，但 1 个月内恢复
- **算力影响**：BTC 算力从 200 EH/s 跌至 100 EH/s（-50%），矿工外迁至美国、哈萨克斯坦
- **结构性影响**：中国 BTC 算力占比从 65% 降至 21%（2024 年）
- **资金外流**：Chainalysis 估计 2021-2022 年超 1000 亿美元加密资产从中国迁出

### 3. 2022 年 2 月俄乌战争

- **2022 年 2 月 24 日**：俄罗斯入侵乌克兰
- 3 天内，乌克兰政府收到 **1 亿美元加密捐款**（BTC、ETH、USDT）
- 乌克兰政府**官方加密钱包地址** Twitter 公布
- 俄罗斯卢布兑美元 24 小时内腰斩 30%
- 俄罗斯公民使用 USDT、USDC 在 P2P 平台兑换外币
- **OFAC 反应**：2022 年 4 月美国禁止美国公民向俄罗斯提供加密支付服务

### 4. 2022 年 3 月美对俄加密制裁

- **OFAC 制裁**俄罗斯最大暗网市场 Hydra（2022 年 4 月）
- **Tornado Cash 制裁**（2022 年 8 月）：OFAC 制裁 Tornado Cash 智能合约地址

  - 2023 年 11 月第五巡回法院裁定 OFAC 越权

### 5. 2023 年 10 月 Hamas 攻击 + 以色列战争

- **Chainalysis 报告**：2023 年与恐怖组织相关的加密地址收到 **1600 万美元**

## A.1 Caldara-Iacoviello GPR 指数

```python
"""
GPR (Geopolitical Risk) Index 复现
- 论文：Caldara & Iacoviello (2022) "Measuring Geopolitical Risk"
- 基于 11 家主要报纸 (NYT, WSJ, WaPo, ...) 头条
- 月度发布
"""
import requests
import pandas as pd
from typing import Optional


GPR_HISTORICAL_URL = "https://www.matteoiacoviello.com/gpr_files/data/gpr_export.csv"


def fetch_gpr_index() -> pd.DataFrame:
    """
    下载 Caldara-Iacoviello GPR 月度数据
    列：month, GPR, GPR_ACT, GPR_THREAT, GPR_HIST
    """
    r = requests.get(GPR_HISTORICAL_URL, timeout=15)
    r.raise_for_status()
    from io import StringIO
    df = pd.read_csv(StringIO(r.text))
    df['date'] = pd.to_datetime(df['month'], format='%Y%m')
    return df.set_index('date')[['GPR', 'GPR_ACT', 'GPR_THREAT', 'GPR_HIST']]


def gpr_spike_threshold(gpr: pd.Series, n_std: float = 2.0) -> pd.Series:
    """GPR > mean + 2σ 视为异常峰值"""
    th = gpr.mean() + n_std * gpr.std()
    return gpr[gpr > th]
```

## A.2 地缘新闻事件分类

```python
"""
地缘事件分类（简化 NLP）
- 类别：war / sanctions / diplomacy / election / disaster / trade
- 用关键词 + 嵌入
"""
import re
from dataclasses import dataclass


EVENT_PATTERNS = {
    'war': r'\b(invasion|war|missile|airstrike|troops|military|coup)\b',
    'sanctions': r'\b(sanction|embargo|blacklist|asset freeze|export ban)\b',
    'diplomacy': r'\b(ceasefire|peace|treaty|summit|bilateral|negotiation)\b',
    'election': r'\b(election|vote|ballot|referendum|candidate|campaign)\b',
    'disaster': r'\b(earthquake|flood|hurricane|typhoon|wildfire)\b',
    'trade': r'\b(tariff|trade war|quota|import duty|export control)\b',
}


@dataclass
class GeopolEvent:
    text: str
    category: str
    severity: int     # 0-10
    countries: list


def classify_event(text: str) -> GeopolEvent:
    text_lower = text.lower()
    matched = []
    for cat, pat in EVENT_PATTERNS.items():
        if re.search(pat, text_lower):
            matched.append(cat)
    primary = matched[0] if matched else 'other'
    severity = min(len(matched) * 3, 10)
    return GeopolEvent(
        text=text, category=primary,
        severity=severity, countries=[])
```

## A.4 战争/冲突 → BTC 影响统计

```python
"""
历史战争事件对 BTC 短期影响（实证）
- 事件：俄乌战争 (2022-02-24), 以哈冲突 (2023-10-07) 等
- 测度：事件前后 7 日 / 30 日 收益与波动
"""
import pandas as pd
import numpy as np


HISTORICAL_EVENTS = [
    {"name": "Russia-Ukraine", "date": "2022-02-24", "btc_pre": 39100, "btc_post7d": 38000, "btc_post30d": 38000},
    {"name": "Israel-Hamas",   "date": "2023-10-07", "btc_pre": 27900, "btc_post7d": 26800, "btc_post30d": 37000},
    {"name": "US-Iran tension","date": "2020-01-03", "btc_pre": 7200,  "btc_post7d": 8200,  "btc_post30d": 9300},
    {"name": "Saudi oil attack","date": "2019-09-14","btc_pre": 10100, "btc_post7d": 10300, "btc_post30d": 9200},
]


def event_returns(events: list = HISTORICAL_EVENTS) -> pd.DataFrame:
    df = pd.DataFrame(events)
    df['ret_7d'] = (df['btc_post7d'] - df['btc_pre']) / df['btc_pre']
    df['ret_30d'] = (df['btc_post30d'] - df['btc_pre']) / df['btc_pre']
    return df[['name', 'date', 'ret_7d', 'ret_30d']]
```

## A.6 地缘情绪评分（NLP）

```python
"""
基于 transformer 的地缘新闻情绪评分
- 模型：FinBERT / ProsusAI/finbert
- 输入：新闻标题 / 正文
- 输出：positive / negative / neutral 概率
"""
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch


class GeopolSentiment:
    def __init__(self):
        self.tok = AutoTokenizer.from_pretrained("ProsusAI/finbert")
        self.m = AutoModelForSequenceClassification.from_pretrained("ProsusAI/finbert")

    def score(self, text: str) -> dict:
        inputs = self.tok(text, return_tensors="pt", truncation=True, max_length=512)
        with torch.no_grad():
            outputs = self.m(**inputs)
        probs = torch.softmax(outputs.logits, dim=-1).numpy()[0]
        return {
            'positive': float(probs[0]),
            'negative': float(probs[1]),
            'neutral':  float(probs[2]),
            'sentiment_index': float(probs[0] - probs[1]),  # -1 to 1
        }
```
