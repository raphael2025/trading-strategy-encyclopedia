## 2. Terra UST 死亡螺旋（详细复盘）

### 2.1 时间线
- **2021-12**：Do Kwon 推出 UST / LUNA
- **2022-01~04**：UST 从 $5B → $18B（市值）
- **2022-04-22**：Anchor 协议调整利率 19% → 18% → 16% → 14%
- **2022-05-07**：攻击者开始抛售 UST 4 亿美元
- **2022-05-08-09**：Curve 池 UST-3Crv 严重失衡
- **2022-05-10**：LUNA 价格从 $80 → $0
- **2022-05-12**：UST 流通量从 188 亿 → 0

### 2.2 数学解释
```
UST 铸造 / 销毁机制：
销毁 1 UST → 铸造价值 1 美元 LUNA
销毁 1 LUNA → 铸造价值 1 美元 UST

若 UST 脱锚 0.98：
套利：买 1 UST = $0.98，销毁换 LUNA = $1 LUNA，卖出
LUNA 抛压 → LUNA 跌 → UST 套利空间更大 → 抛压更大
反向：销毁 LUNA 铸造 UST → UST 流通量增加 → 进一步脱锚
```

### 2.3 教训
1. 算法稳定币 = 银行部分准备金
2. 高收益（Anchor 19%）不可持续
3. 单一稳定币 → 集中风险

## 4. USDe 闪崩（2025-11）

### 4.1 时间线
- **2025-11-08**：ETH 价格 30 分钟闪崩 12%（$3200 → $2820）
- **2025-11-08**：USDe delta-neutral 头寸爆仓
- **2025-11-08**：USDe 跌至 $0.96
- **2025-11-09**：Ethena 启动保险基金
- **2025-11-10**：USDe 回到 $0.998

### 4.2 风险机制
```
USDe 机制：
1 ETH 抵押 + 1 ETH 空单 perp = delta-neutral
净敞口 ≈ 0
但 perp 流动性不足时，空单无法平仓
↓
ETH 暴跌 + perp 流动性枯竭
↓
抵押率不足，触发清算
↓
USDe 持有者面临 depeg
```

### 4.3 教训
1. delta-neutral ≠ 风险中性
2. perp 流动性是关键
3. 保险基金必备
4. 极端行情下 12% 闪崩不可承受

        curve_px = self.check_curve('USDC')
        depeg = min(cex_px, curve_px)
