## 2. USDT（Tether）

- **市值**：>$140B (2026-Q2)
- **链**：Tron (主), Ethereum, Arbitrum, TON, Solana
- **储备**（自我披露）：
  - 美国国债 (T-Bills) ~75%
  - 现金 + 逆回购 ~10%
  - 公司债 / 贵金属 / 比特币 ~15%
- **审计**：未完整审计，仅定期证明 (Proof of Reserves)
- **风险点**：
  - 储备不透明
  - 监管曝光（NYAG, DOJ）
  - Tron 链 USDT 占比 60% → 单一链风险

## 3. USDC（Circle）

- **市值**：>$50B
- **链**：20+ 链（Ethereum, Solana, Base, Arbitrum, OP, Polygon, NEAR, Stellar…）
- **储备**：
  - 短期美国国债 (~80%)
  - 现金 (~20%)
  - BlackRock 托管
- **审计**：Big Four 审计 (Deloitte)，月度储备证明
- **风险点**：
  - SVB 2023 危机：3 天 depeg 至 $0.87
  - 受美国监管直接管辖

## 4. 算法稳定币历史

### 4.1 Terra UST（2022 崩盘）
- 机制：UST ↔ LUNA 双币，1 美元锚定
- 崩盘：Anchor Protocol 19% 收益无法持续 → 死亡螺旋
- 损失：$40B → $0
- 教训：纯算法稳定币不可持续

### 4.2 FRAX（混合）
- USDC 抵押 + 算法
- 现已转 USDC 100% 抵押
- 风险：依赖 USDC → USDC 风险

### 4.3 DAI / MakerDAO
- 多抵押（ETH, RWA）
- RWA 比例 2026：>40%
- Spark Protocol + RWA（国债）
- 风险：RWA 资产质量（贝莱德 BUIDL）

## 6. 储备审计方法

### 6.1 Proof of Reserves (PoR)
- 链上钱包余额快照
- 中心化审计师签名
- 工具：Chainalysis, Nansen, Arkham, RWA.xyz

### 6.2 完整审计（Full Audit）
- USDC：Deloitte 月度审计
- Tether：仅 attestation

### 6.3 第三方追踪
- **DefiLlama Stablecoins**：TVL + 链分布
- **RWA.xyz**：RWA 上链数据
- **Circle Reserves**：实时 USDC 储备

## 8. 关键风险维度

| 维度 | 解释 | 监控指标 |
|---|---|---|
| 储备质量 | T-Bills > 商业票据 > 公司债 | 储备组成月报 |
| 链集中度 | Tron 60% USDT 风险 | 链上余额分布 |
| 监管 | 美国/欧盟牌照 | 注册状态 |
| 赎回延迟 | 1-3 天 | 用户体验 |
| 储备金托管 | BlackRock / BNY Mellon | 托管方信用 |
| 透明度 | 月度 vs 实时 | 报告频率 |

## 9. 交易策略视角

### 9.1 稳定币套利
- **CEX-DEX 价差**：币安 USDT/USDC 折溢价
- **跨链桥费**：Arbitrum → Mainnet USDC 0.1%
- **DeFi 利率差**：Aave USDC 4% vs Compound USDC 3.5%

### 9.2 Depeg 风险溢价
- 历史 depeg：USDC (2023-03, SVB)
- 检测：Chainlink 喂价 < 0.995 → 触发
- 应对：自动切换 USDC → USDT / DAI

## 10. 与 12-2026 关联

- 稳定币 = 机构入场通道
- USDC 是 12-2026 机构方法的首选 stablecoin
- 储备审计 = 机构尽调核心
- Travel Rule = 跨链 stablecoin 必备
