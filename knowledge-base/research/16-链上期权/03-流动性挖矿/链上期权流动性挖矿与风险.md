> 文档版本: 2026-06

## 1. 链上期权 LP 经济学

链上期权 LP（卖方）收入：
1. 权利金收入
2. 协议代币奖励（流动性挖矿）
3. 流动性激励（ve-token 模式）

成本：
1. delta 对冲成本
2. 智能合约风险
3. 极端行情穿仓
4. 机会成本

## 2. Lyra V2 LP 模型

- **机制**：LP 铸造期权（限价单挂单），买方吃单
- **风险参数**：
  - delta 中和阈值
  - max position size
  - volatility surface 范围
- **APY 组成**：
  - 权利金：~15-30%
  - LYRA 奖励：~5-15%
  - ve-LYRA boost：1.0-2.5x
- **典型 APY**：20-60%（年化）

## 3. Aevo 流动性激励

- **机制**：做市商订单簿挂单奖励 AEVO
- **AEVO 释放**：2024-03
- **ve-AEVO 模型**：质押 + boost
- **做市商 tier**：
  - Tier 1: >$10M maker 深度
  - Tier 2: $1M-$10M
  - Tier 3: <$1M

## 4. Dopex SSOV 模式

- **Single Staking Options Vault**：
  - 用户质押 ETH/USDC/DPX
  - SSOV 卖出 covered call/put
  - 收益分配：
    - 70% 卖方（用户）
    - 20% 协议
    - 10% 质押者
- **典型 APY**：
  - covered call: 15-40%
  - cash-secured put: 10-25%
  - 牛市偏低

## 5. Hegic V2 LP

- **机制**：单一 LP 池
- **风险**：高（已被攻击 1 次）
- **V2 改进**：
  - 多池
  - 限价触发
  - Vouchers (NFT 期权凭证)

## 6. Premia Blue

- **机制**：CLAMM（集中流动性 AMM）
- **特点**：
  - 集中流动性在 strike 附近
  - 减少 IL
  - 卖方单边做市

## 7. Panoptic（无期限期权）

- **机制**：永续期权（perpetual options）
- **关键创新**：
  - Uniswap V3 风格 LP
  - 永续美式期权
  - 0 IL（理论上）
- **风险**：
  - gamma 损失
  - 预言机风险

## 8. 风险维度详解

### 8.2 价格偏离
- 链上 IV 偏 CEX 5-20%
- 大单滑点 >5%
- 对手方风险（清算人押注）

### 8.3 流动性风险
- LP 撤出 → 池子深 <$1M
- 大单无法成交
- 跨链桥时间

### 8.4 监管风险
- 期权 = 证券（SEC 视角）
- 协议 = 未注册交易所
- 美国用户屏蔽

## 11. Python：LP 收益回测

```python
"""
链上期权 LP 收益回测
- 历史波动率
- 对冲成本
- 净 APY
"""
import numpy as np
import pandas as pd


def backtest_lp_pnl(prices: pd.Series, iv: pd.Series,
                    T: float = 7/365, K_pct: float = 1.1,
                    hedge_freq_h: int = 4) -> pd.DataFrame:
    rows = []
    pos = 0
    cost_basis = 0
    for i in range(0, len(prices), 24):  # 每日
        S = prices.iloc[i]
        K = S * K_pct
        sigma = iv.iloc[i]
        # 卖出 call
        call_px = bs_call(S, K, T, 0.05, sigma)
        pos += 1
        cost_basis -= call_px
        # delta 对冲
        hedge_pnl = 0
        for j in range(0, 24, hedge_freq_h):
            if i + j + hedge_freq_h >= len(prices):
                break
            dS = prices.iloc[i + j + hedge_freq_h] - prices.iloc[i + j]
            delta = bs_greeks(S, K, T, 0.05, sigma)['delta']
            hedge_pnl -= delta * dS
        # 期末 close
        S_final = prices.iloc[min(i + 24, len(prices) - 1)]
        intrinsic = -max(S_final - K, 0)
        rows.append({
            "date": prices.index[i],
            "premium": call_px,
            "intrinsic": intrinsic,
            "hedge_pnl": hedge_pnl,
            "net_pnl": call_px + intrinsic + hedge_pnl
        })
    return pd.DataFrame(rows)
```

## 12. Python：ve-token boost 模拟

```python
"""
Lyra 风格 ve-token boost 计算
"""
def calc_ve_boost(ve_token_lock_days: int, max_days: int = 730,
                   base_boost: float = 1.0) -> float:
    return base_boost + (ve_token_lock_days / max_days) * 1.5
```
