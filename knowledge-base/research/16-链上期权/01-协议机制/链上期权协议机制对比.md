# 链上期权协议机制对比

> 文档版本: 2026-06
> 引用: 22 个
> 知识截止: 2026-05-31

## 1. 链上期权四代演进

```
1.0 (2019-2020): Hegic, Auctus    — 单一做市商 (CLOB-like OTC)
2.0 (2020-2021): Opyn, Primitive  — 拍卖 / 集中流动性
3.0 (2021-2022): Lyra, Premia     — AMM 风格 + Black-Scholes 定价
4.0 (2022+):     Aevo, Ribbon      — 永续 + 限价 + 结构化
5.0 (2025+):     Zeta, Lyra V2     — 跨链 + 组合 + 链上 CLOB
```

## 2. Hegic（先驱，单一做市商）

- **机制**：单一 LPs 提供流动性，用户按需铸造期权
- **定价**：Black-Scholes 离线计算
- **风险**：2021-04 攻击损失 $48M
- **现状**：v2 重启，更安全
- **TVL**：<$30M

## 3. Lyra（前 3.0 代表）

- **机制**：AMM 风格期权池
- **定价**：Black-Scholes + SABR 波动率曲面
- **特性**：
  - 流动性提供者 = 期权卖方
  - 自动 delta 中和（delta hedging）
  - 损失 vs 收益清晰
- **Lyra V2 (2024)**：
  - 跨链部署
  - 机构级流动性
- **TVL**：>$80M

## 4. Aevo（高性能 + 限价）

- **机制**：订单簿 + 永续（off-chain settlement，链上结算）
- **特性**：
  - 限价期权
  - 永续合约（同账户）
  - 跨保证金
- **Aevo EVM L2**（OP Stack 链）：
  - 衍生品专用 L2
- **TVL**：>$150M

## 5. Ribbon（结构化期权）

- **机制**：自动化期权策略（covered call, put spread）
- **v1 风险**：2022-10 攻击损失 $5M
- **v2 进化**：
  - 集成 Aevo
  - 自动化策略（Theta Vault）
- **TVL**：>$100M

## 6. Premia（多模式）

- **机制**：orderbook + AMM
- **特性**：
  - 现货 + 永续 + 期权
  - 跨链（Arbitrum, Optimism, Base, Fantom）
  - Pools 系统
- **TVL**：>$20M

## 7. Dopex（双代币模型）

- **机制**：双代币 LP (rDPX + DOPEX)
- **特性**：
  - 欧式期权
  - 闪电对冲
  - Atlantic Options
- **Dopex V2 (SSOV)**：
  - Single Staking Options Vault
  - 集中流动性
- **TVL**：>$50M

## 8. Zeta Markets（Solana）

- **机制**：链上 CLOB + perp
- **特性**：
  - 跨保证金
  - BTC / SOL 期权
- **v2 (2025)**：机构级 API

## 9. Lyra V2（2024 重启）

- **架构**：
  - 订单簿链下 + 链上结算
  - 跨链（Base + OP）
  - 做市商激励
- **市场规模**：>$200M 月交易量

## 11. 关键术语

- **Greeks**：delta, gamma, vega, theta, rho（详见 05-02）
- **Implied Volatility (IV)**：市场隐含波动率
- **DVOL**：Deribit 波动率指数（行业基准）
- **Skew**：OTM put IV - OTM call IV
- **SABR**：随机波动率模型
- **ATM / OTM / ITM**：行权价关系
- **Exotic**：障碍期权、二元、永续期权
