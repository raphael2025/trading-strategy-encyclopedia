## 🔑 核心心法

1. **保护本金 > 赚取利润**
2. **截断亏损，让利润奔跑**
3. **等待完美的入场**（75% 时间等待）
4. **承认错误**
5. **一致性**（同策略、同仓位）
6. **跟随趋势**
7. **关注过程而非结果**
8. **多时间框架分析**
9. **严格风险管理**
10. **避免过度优化**
